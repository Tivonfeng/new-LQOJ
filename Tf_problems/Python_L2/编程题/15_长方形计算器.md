# 长方形计算器

## 题目描述

请编写程序求一个长方形周长和面积的计算器。

---

## 要求

1. 用户从控制台输入长和宽（有可能是小数，单位是厘米）；
2. 如果用户输入的数不大于0（不考虑浮点数之外的数据类型），显示信息："输入有误，请重新输入！";
3. 输出长方形的周长和面积，结果保留两位小数，并注明哪个是周长，哪个是面积。

---

## 提示

- 周长公式：周长 = (长 + 宽) × 2
- 面积公式：面积 = 长 × 宽

---

## 参考代码

```python
length = float(input("请输入长："))
width = float(input("请输入宽："))
if length <= 0 or width <= 0:
    print("输入有误，请重新输入！")
else:
    perimeter = 2 * (length + width)
    area = length * width
    print(f"周长是：{perimeter:.2f}")
    print(f"面积是：{area:.2f}")
```

---

## 考点

- 数学计算 - 几何公式应用
- 浮点数输入和格式化输出
