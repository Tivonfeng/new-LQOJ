# 小学生体能测试

## 题目描述

学校要进行小学生体能测试。根据学生的性别和年级，需要分配不同的测试项目。规则如下：

| 年级 | 性别 | 测试项目 |
|------|------|----------|
| 1-3  | 男   | 引体向上 |
| 1-3  | 女   | 仰卧起坐 |
| 4-6  | 男   | 800米跑步 |
| 4-6  | 女   | 500米跑步 |

请编写一个Python程序，根据输入的学生性别（"男"或"女"）和年级（1到6的整数），输出该学生需要参加的体测项目。

---

## 输入要求

1. 输入两行：第一行输入字符串，代表学生的性别，用"男"或"女"表示；
2. 第二行输入数字，代表学生的年级，用1到6的整数表示。

---

## 输出要求

输出一行字符串，输出学生需要参加的体测项目。

---

## 样例输入

```
女
1
```

## 样例输出

```
仰卧起坐
```

---

## 参考代码

```python
# 读取输入
gender = input().strip()
grade = int(input().strip())

# 判断测试项目
if 1 <= grade <= 3:
    if gender == '男':
        print('引体向上')
    elif gender == '女':
        print('仰卧起坐')
elif 4 <= grade <= 6:
    if gender == '男':
        print('800米跑步')
    elif gender == '女':
        print('500米跑步')
```

---

## 考点

- 条件判断 - 多分支组合
- 年级(1-3/4-6) × 性别(男/女) 交叉判断
