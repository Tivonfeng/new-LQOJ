# 自动分拣土豆机

## 题目描述

马铃薯也叫土豆，是一种常见的根茎类蔬菜，每个土豆根据重量可以分为以下四个等级：

- **A类（大土豆）**：重量 ≥ 50克
- **B类（中土豆）**：50克 > 重量 ≥ 30克
- **C类（小土豆）**：30克 > 重量 ≥ 20克
- **D类（微小土豆）**：重量 < 20克

不同类别市场批发价格也不同：
- A类：1.5元/500克
- B类：1.2元/500克
- C类：1元/500克
- D类：0.7元/500克

阿宝下乡研学期间发现土豆的分类和分拣通常由人工观察，进行大致分类。阿宝想，是否可以做一个自动分拣土豆机，根据土豆的重量结合机械手臂进行自动分拣。

---

## 要求

1. 录入每个土豆的重量（用来模拟称重后的读数）；假定输入数据是合理范围（都是大于等于0的数字，为了便于计算，要求输入的数据都是整数）。
2. 当录入0时，程序停止运行，并打印所有土豆售出后的销售额；（提醒大家，土豆的售价是以500克为单位，每个土豆的价格需要进行换算，换算公式为：每个土豆价格 = 每个土豆的重量 × 批发价 / 500）。
3. 销售金额，最终结果保留2位小数。

---

## 输入输出样例

**输入样例：**
```
80
50
90
70
40
30
20
10
20
60
78
65
0
```

**输出样例：**
```
销售额是：1.74元
```

---

## 参考代码

```python
prices = []
while True:
    weight = int(input())
    if weight == 0:
        break
    # 根据重量判断类别并计算价格
    if weight >= 50:
        price = weight * 1.5 / 500
    elif weight >= 30:
        price = weight * 1.2 / 500
    elif weight >= 20:
        price = weight * 1.0 / 500
    else:
        price = weight * 0.7 / 500
    prices.append(price)

total = sum(prices)
print(f"销售额是：{total:.2f}元")
```
