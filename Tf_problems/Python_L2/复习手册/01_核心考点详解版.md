## 一、变量和数据类型

### 1.1 Python基本数据类型

| 类型 | 说明 | 示例 |
|------|------|------|
| `int` | 整数 | `10`, `-5`, `0` |
| `float` | 浮点数 | `3.14`, `-0.5`, `1.0` |
| `str` | 字符串 | `"hello"`, `'Python'` |
| `bool` | 布尔值 | `True`, `False` |
| `list` | 列表 | `[1, 2, 3]`, `['a', 'b']` |
| `tuple` | 元组 | `(1, 2, 3)`, `('a', 'b')` |
| `dict` | 字典 | `{'name': 'Tom', 'age': 18}` |

### 1.2 变量命名规则

```python
# 合法的变量名
name = "Tom"           # ✅ 以字母开头
_age = 18              # ✅ 以下划线开头
myVar123 = "Python"    # ✅ 包含字母、数字、下划线

# 非法的变量名
2nd = "second"         # ❌ 以数字开头
my-var = "test"        # ❌ 包含连字符
my var = "test"        # ❌ 包含空格
```

### 1.3 类型转换函数

```python
# 常用类型转换
int("123")     # 字符串转整数 → 123
float("3.14")  # 字符串转浮点数 → 3.14
str(123)       # 整数转字符串 → "123"
bool(0)        # 布尔值转换 → False
bool("")       # 空字符串 → False
bool([])       # 空列表 → False
```

### 1.4 input()函数特性

```python
# input()返回的一定是字符串类型！
age = int(input("请输入年龄:"))    # 需要int()转换
score = float(input("请输入分数:")) # 需要float()转换

# 错误示例
num = input()       # 输入"10"，num是字符串"10"，不是整数10
```

### 1.5 经典真题

**【真题2303-12】** 下列说法正确的是？
- A. 字典只能存储字符串，不能存储其他类型的对象
- B. `dict1={}`可以创建一个空字典
- C. `(123:456)`是一个合法的字典
- D. 字典的值必须是唯一的，键不必是唯一的

**答案**：B

**解析**：
- A：错误，字典可以存储任意类型的对象
- B：正确，`{}`创建空字典
- C：错误，元组用()`，字典用`{}`
- D：错误，字典的键必须唯一，值可以重复

## 二、列表操作

### 2.1 列表创建与基本操作

```python
# 创建列表
list1 = [1, 2, 3, 4, 5]              # 直接创建
list2 = list(range(5))               # range创建 → [0,1,2,3,4]
list3 = []                           # 空列表
list4 = ['a', ['b', 'c'], 'd']      # 嵌套列表

# 访问元素（索引从0开始）
list1[0]     # 第一个元素 → 1
list1[-1]    # 最后一个元素 → 5
list1[1:3]   # 切片（索引1到2）→ [2, 3]
list1[:3]    # 前3个元素 → [1, 2, 3]
list1[::2]   # 步长2 → [1, 3, 5]
```

### 2.2 列表方法详解

| 方法 | 功能 | 示例 | 结果 |
|------|------|------|------|
| `append()` | 末尾添加单个元素 | `[1,2].append(3)` | `[1,2,3]` |
| `insert(i, x)` | 在索引i处插入 | `[1,2].insert(1, 9)` | `[1,9,2]` |
| `remove(x)` | 删除第一个x | `[1,2,2].remove(2)` | `[1,2]` |
| `pop()` | 删除末尾元素 | `[1,2].pop()` | `[1]` |
| `pop(i)` | 删除索引i元素 | `[1,2,3].pop(1)` | `[1,3]` |
| `len()` | 元素个数 | `len([1,2,3])` | `3` |
| `index(x)` | 查找x首次位置 | `[1,2,3].index(2)` | `1` |
| `count(x)` | 统计x出现次数 | `[1,2,2].count(2)` | `2` |
| `sort()` | 排序（原地修改） | `[3,1,2].sort()` | `[1,2,3]` |
| `sorted()` | 返回排序新列表 | `sorted([3,1,2])` | `[1,2,3]` |

### 2.3 列表复制注意事项

```python
# 错误：浅拷贝（指向同一对象）
list1 = [1, 2, 3]
list2 = list1
list2.append(4)
print(list1)  # [1, 2, 3, 4]  ← list1也被修改！

# 正确：创建新列表
list2 = list1[:]       # 切片复制
list2 = list1.copy()   # copy方法
list2 = list(list1)    # 工厂函数
```

### 2.4 列表与表达式

```python
# 列表重复运算符
[1, 2] * 2   # → [1, 2, 1, 2]

# 列表连接
[1, 2] + [3, 4]  # → [1, 2, 3, 4]

# 嵌套列表长度
[['a', 'b'], ['c', 'd']]  # len()返回2（外层2个元素）
```

### 2.5 经典真题

**【真题2306-5】** 运行以下程序，输出的结果是？
```python
numbers=[1,2,3,4]
numbers.append([5,6,7,8])
print(len(numbers))
```
- A. 4
- B. 5
- C. 8
- D. 12

**答案**：B

**解析**：`append()`将整个列表作为单个元素添加，`[5,6,7,8]`是一个元素，总共5个元素。

**【真题2312-2】** 执行`nums.append(nums[4])`的结果是？
```python
nums = [1,2,3,4]
nums.append(nums[4])
```
- A. [1,2,3,4,5]
- B. [1,2,3,4,4]
- C. [1,2,3,4]
- D. 报错

**答案**：D

**解析**：`nums[4]`超出索引范围（0-3），会触发IndexError。

## 三、字典操作

### 3.1 字典创建方式

```python
# 方式1：字面量
dict1 = {'name': 'Tom', 'age': 18}

# 方式2：dict()函数
dict2 = dict(name='Jerry', age=20)  # {'name': 'Jerry', 'age': 20}

# 方式3：空字典
dict3 = {}
dict4 = dict()

# 嵌套字典
dict5 = {
    'user1': {'name': 'Alice', 'score': 95},
    'user2': {'name': 'Bob', 'score': 88}
}
```

### 3.2 字典基本操作

```python
d = {'name': 'Tom', 'age': 18, 'score': 95}

# 访问值（通过键）
d['name']     # → 'Tom'
d.get('name') # → 'Tom'（推荐，键不存在返回None）
d.get('height', 170)  # → 170（默认值）

# 修改值
d['age'] = 19  # → {'name': 'Tom', 'age': 19, 'score': 95}

# 添加键值对
d['gender'] = 'M'  # → 新增
d['score'] = 100   # → 修改

# 删除键值对
del d['gender']     # 删除指定键
d.pop('age')       # 删除并返回值
d.clear()          # 清空字典
```

### 3.3 字典方法

| 方法 | 功能 | 示例 | 结果 |
|------|------|------|------|
| `keys()` | 返回所有键 | `d.keys()` | `dict_keys(['name', 'age'])` |
| `values()` | 返回所有值 | `d.values()` | `dict_values(['Tom', 18])` |
| `items()` | 返回键值对 | `d.items()` | `dict_items([('name', 'Tom'), ...])` |
| `get(k)` | 获取值 | `d.get('name')` | `'Tom'` |
| `get(k, v)` | 默认值 | `d.get('x', 0)` | `0` |
| `pop(k)` | 删除并返回值 | `d.pop('age')` | `18` |
| `update()` | 合并字典 | `d.update({'x': 1})` | 合并 |

### 3.4 遍历字典

```python
d = {'name': 'Tom', 'age': 18, 'score': 95}

# 遍历键
for key in d.keys():
    print(key)

# 遍历值
for value in d.values():
    print(value)

# 遍历键值对
for key, value in d.items():
    print(f"{key}: {value}")
```

### 3.5 经典真题

**【真题2312-16】** 从字典里获取"西瓜"
```python
name_dict = {1:'西瓜', 2:'南瓜'}
```

- D. `name_dict[2]`

**解析**：字典通过键访问值，"西瓜"对应的键是1。

## 四、元组操作

### 4.1 元组创建

```python
# 基本创建
t1 = (1, 2, 3)           # 用圆括号
t2 = 1, 2, 3             # 逗号分隔（隐式元组）
t3 = (5,)                # 单元素元组（必须有逗号）
t4 = ()                  # 空元组

# 创建单元素元组
t5 = (5)     # ❌ 这是整数5，不是元组
t6 = (5,)    # ✅ 正确的单元素元组

# 混合类型
t7 = ('a', 1, True, [1, 2])

# 元组可嵌套
t8 = ((1, 2), (3, 4))
```

### 4.2 元组特点

```python
# 不可变性
t = (1, 2, 3)
t[0] = 100    # ❌ TypeError：元组不可修改
t.append(4)  # ❌ AttributeError

# 可哈希（可以作为字典键）
d = {(1, 2): 'point'}  # ✅ 合法
d = {[1, 2]: 'list'}   # ❌ 列表不可哈希
```

### 4.3 元组操作

| 操作 | 功能 | 示例 | 结果 |
|------|------|------|------|
| 索引访问 | 获取元素 | `t[0]`, `t[-1]` | 元素值 |
| 切片 | 获取子元组 | `t[1:3]` | `(2, 3)` |
| 连接 | 合并元组 | `(1,2) + (3,4)` | `(1,2,3,4)` |
| 重复 | 重复元组 | `(1,2) * 2` | `(1,2,1,2)` |
| `len()` | 元素个数 | `len((1,2,3))` | `3` |
| `index(x)` | 查找位置 | `(1,2,3).index(2)` | `1` |
| `count(x)` | 统计次数 | `(1,2,2).count(2)` | `2` |

### 4.4 元组与列表转换

```python
t = (1, 2, 3)
lst = list(t)   # 元组转列表 → [1, 2, 3]

lst = [4, 5, 6]
t = tuple(lst)  # 列表转元组 → (4, 5, 6)
```

### 4.5 经典真题

**【真题2312-3】** 下列哪个是元组？
- A. `[1,2,3]`
- B. `(1,2,3)`
- C. `[1,2,3]`
- D. `(1:小明,2:小李)`

**答案**：B

**解析**：元组用圆括号`()`，键值对是字典格式。

**【真题2309-6】** 元组方法
```python
t=('s','a','a','r',5)
print(t.index('a') + t.count('a'))
```
- A. 2
- B. 3
- C. 4
- D. 5

**答案**：B

**解析**：`index('a')`返回第一个'a'的索引（1），`count('a')`返回出现次数（2），1+2=3。

## 五、字符串操作

### 5.1 字符串基本操作

```python
s = "Hello Python"

# 索引和切片
s[0]           # → 'H'
s[-1]          # → 'n'
s[0:5]         # → 'Hello'
s[6:]          # → 'Python'
s[::-1]        # → 'nohtyP olleH'（反转）

# 常用操作
len(s)         # 长度 → 12
"Hello" in s   # 子串检查 → True
```

### 5.2 字符串方法

| 方法 | 功能 | 示例 | 结果 |
|------|------|------|------|
| `upper()` | 转大写 | `"abc".upper()` | `'ABC'` |
| `lower()` | 转小写 | `"ABC".lower()` | `'abc'` |
| `split()` | 分割 | `"a,b,c".split(",")` | `['a','b','c']` |
| `join()` | 连接 | `",".join(['a','b'])` | `'a,b'` |
| `replace()` | 替换 | `"a b".replace(" ", ",")` | `'a,b'` |
| `strip()` | 去除空白 | `" ab ".strip()` | `'ab'` |
| `find()` | 查找位置 | `"abc".find("b")` | `1` |
| `index()` | 查找位置 | `"abc".index("b")` | `1` |

### 5.3 字符串格式化

```python
# format方法
"{} {}".format("Hello", "Python")                    # → 'Hello Python'
"{1} {0}".format("Python", "Hello")                 # → 'Hello Python'
"{name} is {age} years old".format(name='Tom', age=18)

# f-string（Python 3.6+）
name = 'Tom'
age = 18
f"{name} is {age} years old"                        # → 'Tom is 18 years old'

# %格式化（旧式）
"%s is %d years old" % ('Tom', 18)                  # → 'Tom is 18 years old'
```

### 5.4 字符串不可变

```python
s = "Hello"
s[0] = 'h'    # ❌ TypeError

# 正确做法：创建新字符串
s = "Hello"
s_new = 'h' + s[1:]  # → 'hello'
```

## 六、循环结构

### 6.1 for循环

```python
# 遍历列表
for item in [1, 2, 3]:
    print(item)

# 遍历字符串
for char in "abc":
    print(char)

# range函数
for i in range(5):      # 0,1,2,3,4
for i in range(1, 5):   # 1,2,3,4
for i in range(1, 5, 2): # 1,3（步长2）
```

### 6.2 while循环

```python
# 基本结构
i = 0
while i < 5:
    print(i)
    i += 1

# 死循环（需要break退出）
while True:
    cmd = input("输入(q退出):")
    if cmd == 'q':
        break
```

### 6.3 break与continue

```python
# break：跳出整个循环
for i in range(10):
    if i == 5:
        break
    print(i)  # 输出 0,1,2,3,4

# continue：跳过本次循环，继续下一次
for i in range(5):
    if i == 2:
        continue
    print(i)  # 输出 0,1,3,4
```

### 6.4 循环控制语句

| 语句 | 功能 |
|------|------|
| `break` | 跳出整个循环体 |
| `continue` | 跳过本次循环剩余代码，进入下一次循环 |
| `else` | 循环正常结束后执行 |

```python
# for-else结构
for i in range(3):
    print(i)
else:
    print("循环正常结束")  # 会执行

# break时else不执行
for i in range(3):
    if i == 1:
        break
    print(i)
else:
    print("不会执行")
```

### 6.5 经典真题

**【真题2312-13】** while循环边界
```python
i=1
while i < 10:
    i += 1
    print(i)
```
- A. 1
- B. 9
- C. 10
- D. 11

**答案**：C

**解析**：i从1开始，当i=9时仍满足条件，i变为10后打印，最终输出10。

**【真题2309-9】** 关于循环的说法

- D. break和continue可用于for和while循环

**答案**：C（错误选项）

## 七、条件判断

### 7.1 if-elif-else结构

```python
# 单分支
if 条件:
    语句块

# 双分支
if 条件:
    语句块1
else:
    语句块2

# 多分支
if 条件1:
    语句块1
elif 条件2:
    语句块2
elif 条件3:
    语句块3
else:
    语句块4
```

### 7.2 逻辑运算符

| 运算符 | 说明 | 示例 | 结果 |
|--------|------|------|------|
| `and` | 逻辑与 | `True and False` | `False` |
| `or` | 逻辑或 | `True or False` | `True` |
| `not` | 逻辑非 | `not True` | `False` |

### 7.3 比较运算符

```python
==  # 等于（值比较）
!=  # 不等于
>   # 大于
<   # 小于
>=  # 大于等于
<=  # 小于等于

# 身份运算符
is      # 同一对象
is not  # 不同对象

# 成员运算符
in      # 在序列中
not in  # 不在序列中
```

### 7.4 条件表达式

```python
# 假值（等价于False）
False, None, 0, 0.0, "", [], {}, ()

# 真值
True, 1, -1, "text", [1], {1:1}, (0,)
```

### 7.5 经典真题

**【真题2312-24】** 嵌套if判断
```python
x = int(input())
if x!=0:
    if x>0:

    y=0
```

输入10，y的值是？
- A. 0
- B. 1
- C. -1
- D. 10

```python
score = eval(input("请输入成绩:"))
```

输入60，输出是？

## 九、综合应用

### 9.1 数据结构选择

| 场景 | 推荐数据结构 | 原因 |
|------|-------------|------|
| 有序集合，可修改 | 列表 `list` | 支持索引、切片、增删 |

### 9.2 算法思想

```python
# 冒泡排序
lst = [3, 7, 2, 1]
for i in range(len(lst)-1):
    for j in range(len(lst)-1-i):
        if lst[j] > lst[j+1]:
            lst[j], lst[j+1] = lst[j+1], lst[j]
# 结果: [1, 2, 3, 7]

# 累加求和
s = 0
for i in range(1, 101):
    s += i

# 查找元素
def find_element(lst, target):
    for i, item in enumerate(lst):
        if item == target:
            return i
    return -1
```
