# 寻找数字

## 题目描述
输入 $t$ 组测试数据，每组一个正整数 $a$，判断是否存在正整数 $b$ 满足 $a = b^4$，存在输出 $b$，否则输出 $-1$。

## 输入格式
第一行正整数 $t$（$1 \leq t \leq 10^5$），后续 $t$ 行各一个正整数（$1 \leq a \leq 10^8$）。

## 输出格式
输出 $b$ 或 $-1$，每个结果占一行。

## 样例

```input1
3
16
81
10
```

```output1
2
3
-1
```

### 样例解释
- $16 = 2^4$，所以输出 $2$
- $81 = 3^4$，所以输出 $3$
- $10$ 不是任何整数的四次方，所以输出 $-1$

## 数据范围
$1 \leq t \leq 10^5$，$1 \leq a \leq 10^8$

## 学习提示
- 可以使用 `sqrt` 函数嵌套：$b = \sqrt{\sqrt{a}}$
- 也可以使用 `pow(a, 0.25)` 计算
- 验证 $b^4$ 是否等于 $a$，注意边界情况
- 使用整数运算避免浮点误差

## 学习目标
- 掌握完全四次方的判断方法
- 理解平方根和四次方根的计算
- 学会处理大数运算的边界情况
