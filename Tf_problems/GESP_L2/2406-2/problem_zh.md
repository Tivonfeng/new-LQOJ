# 计数

## 🎯 问题描述
统计从1到n的所有正整数中，数字k出现的次数（1≤k≤9，1≤n≤1000）。

## 📝 任务要求
给定正整数n和数字k（1-9），统计1到n的所有数字中，k这个数字一共出现了多少次。

## ⬇️ 输入格式
第一行正整数n，第二行正整数k。

## ⬆️ 输出格式
输出k出现的次数。

## 📤 输出示例
```input1
25
2
```

```output1
9
```

## 💡 学习提示
- 遍历从1到n的所有数字
- 对每个数字，分解其每一位
- 统计每位数字等于k的次数
- 累加所有数字中的出现次数

## 🎯 学习目标
- 掌握数位分解的方法
- 理解循环嵌套的应用
- 学会统计数字出现次数的思路
