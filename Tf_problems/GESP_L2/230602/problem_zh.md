# 自幂数判断

## 题目描述
自幂数是指 N 位数的各位数字 N 次方之和等于本身。输入 M 个正整数，判断每个数是否为自幂数。

## 输入格式
第一行正整数 M（$1 \leq M \leq 100$），后续 M行各一个正整数（小于 $10^8$）。

## 输出格式
每个数对应输出 'T'（是）或 'F'（否）。

## 样例

```input1
3
152
111
153
```

```output1
F
F
T
```

```input2
5
8208
548834
88593477
12345
5432
```

```output2
T
T
T
F
F
```

## 数据范围
$1 \leq M \leq 100$，每个正整数 $< 10^8$

## 学习提示
- 先计算数字的位数（可以用循环除 10 来实现）
- 分别计算各位数字的位数次方之和
- 比较计算结果与原数是否相等

## 学习目标
- 掌握数位分解的方法
- 理解幂运算在数位处理中的应用
- 学会将数学概念转化为代码逻辑
