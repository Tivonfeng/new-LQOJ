# 画三角形

## 🎯 问题描述
输入正整数n，用大写字母拼成三角形图案。第1行1个字母，第2行2个字母……由上至下、由左至右依次用A-Z填充，Z之后从头用A填充。

## 📝 任务要求
根据输入的正整数n，输出符合要求的三角形图案，每行右侧无多余空格。

## ⬇️ 输入格式
一行正整数n（2≤n≤40）。

## ⬆️ 输出格式
输出n行三角形图案，每行字母数等于行号。

## 📤 输出示例
```input1
3
```

```output1
A
BC
DEF
```

```input2
7
```

```output2
A
BC
DEF
GHIJ
KLMNO
PQRSTU
VWXYZAB
```

## 💡 学习提示
- 使用双重循环：外层循环控制行数，内层循环控制每行的字母数
- 利用ASCII码转换：'A' + offset 可以得到对应的字母
- 使用取模运算 %26 实现循环填充A-Z

## 🎯 学习目标
- 掌握双重循环的嵌套使用
- 理解ASCII字符与整数的转换关系
- 学会使用取模运算实现循环效果
