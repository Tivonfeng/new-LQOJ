# 找素数

## 🎯 问题描述
找出两个正整数A和B之间（包括A和B）的素数个数。素数是大于1的正整数，除了1和自身外不能被其他正整数整除。

## 📝 任务要求
统计A到B之间（含）的素数数量并输出。

## ⬇️ 输入格式
2行，第一行正整数A，第二行正整数B（2≤A≤B≤1000）。

## ⬆️ 输出格式
输出素数个数C。

## 📤 输出示例
```input1
2
10
```

```output1
4
```

```input2
98
100
```

```output2
0
```

## 💡 学习提示
- 使用循环遍历每个待检查的数
- 对每个数，检查从2到其平方根之间的所有整数是否能整除它
- 素数判断函数可以封装为独立函数

## 🎯 学习目标
- 掌握素数的判定方法
- 理解平方根优化原理
- 学会封装通用函数
