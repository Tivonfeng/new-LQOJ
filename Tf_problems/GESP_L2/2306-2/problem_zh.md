# 自幂数判断

## 🎯 问题描述
自幂数是指N位数的各位数字N次方之和等于本身。输入M个正整数，判断每个数是否为自幂数。

## 📝 任务要求
对每个输入的正整数，判断它是否为自幂数，输出'T'（是）或'F'（否）。

## ⬇️ 输入格式
第一行正整数M（1≤M≤100），后续M行各一个正整数（小于10⁸）。

## ⬆️ 输出格式
每个数对应输出'T'或'F'，可输入一个判断一个输出一个。

## 📤 输出示例
```input1
3
152
111
153
```

```output1
F
F
T
```

```input2
5
8208
548834
88593477
12345
5432
```

```output2
T
T
T
F
F
```

## 💡 学习提示
- 先计算数字的位数（可以用循环除10来实现）
- 分别计算各位数字的位数次方之和
- 比较计算结果与原数是否相等

## 🎯 学习目标
- 掌握数位分解的方法
- 理解幂运算在数位处理中的应用
- 学会将数学概念转化为代码逻辑
