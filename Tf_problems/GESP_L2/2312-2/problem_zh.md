# 小杨的H字矩阵

## 🎯 问题描述
构造N×N的H字矩阵（N为奇数），最左列、最右列是|，中间一行（第(N+1)/2行）的第2~N-1个字符是-，其余为小写字母a。

## 📝 任务要求
根据输入的奇数N，输出N行N列的H字矩阵。

## ⬇️ 输入格式
一行整数N（5≤N≤49，奇数）。

## ⬆️ 输出格式
输出N行矩阵，每行N个字符，无额外符号。

## 📤 输出示例
```input1
5
```
```output1
|aaa|
|aaa|
|---|
|aaa|
|aaa|
```
```input2
7
```
```output2
|aaaaa|
|aaaaa|
|---|---|
|aaaaa|
|aaaaa|
|aaaaa|
```

## 💡 学习提示
- 使用双重循环遍历矩阵的每个位置
- 根据行列位置判断应该输出什么字符
- 关键位置：第一列、最后一列、中间行、中间列
- 注意字符的边界条件处理

## 🎯 学习目标
- 掌握二维数组/矩阵的遍历方法
- 理解多条件判断的组合逻辑
- 学会处理边界条件
