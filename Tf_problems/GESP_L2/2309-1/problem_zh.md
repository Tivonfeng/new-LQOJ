# 买文具

## 🎯 问题描述
签字笔2元一支、记事本5元一本、直尺3元一把，输入小明购买的数量及手中的钱，判断是否够买并输出剩余或缺少的钱。

## 📝 任务要求
根据购买数量和手中的钱，计算总价并判断是否够买，输出"Yes"和剩余钱数或"No"和缺少的钱数。

## ⬇️ 输入格式
输入4行，分别为签字笔、记事本、直尺的数量（1≤X,Y,Z≤10）和手中的钱数。

## ⬆️ 输出格式
够买输出"Yes"和剩余钱数，否则输出"No"和缺少的钱数。

## 📤 输出示例
```input1
1
1
1
20
```

```output1
Yes 10
```

```input2
1
1
1
5
```

```output2
No 5
```

## 💡 学习提示
- 先计算总价：X*2 + Y*5 + Z*3
- 比较总价与手中的钱
- 使用条件判断选择输出内容

## 🎯 学习目标
- 掌握基本算术运算
- 理解条件判断的应用
- 学会处理多个输入的情况
