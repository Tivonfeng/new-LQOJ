# 数三角形

## 题目描述
直角边长 $a$、$b$ 均不超过 $n$ 的正整数，求不同的面积为整数的直角三角形数量（$a,b$ 和 $b,a$ 视为同一个）。

直角三角形的面积为 $\frac{a \times b}{2}$，当 $a \times b$ 为偶数时，面积为整数。

## 输入格式
输入一行整数 $n$（$1 \leq n \leq 1000$）。

## 输出格式
输出三角形数量。

## 样例

```input1
3
```

```output1
3
```

### 样例解释
当 $n = 3$ 时，满足条件的直角三角形有：
- $(1,1)$：面积 $0.5$（不是整数）
- $(1,2)$：面积 $1$（是整数）
- $(1,3)$：面积 $1.5$（不是整数）
- $(2,2)$：面积 $2$（是整数）
- $(2,3)$：面积 $3$（是整数）
- $(3,3)$：面积 $4.5$（不是整数）

由于 $(1,2)$ 和 $(2,1)$ 视为同一个，所以有 3 个不同的三角形。

```input2
5
```

```output2
9
```

## 数据范围
$1 \leq n \leq 1000$

## 学习提示
- 使用双重循环遍历 $a$ 和 $b$，其中 $a$ 从 1 到 $n$，$b$ 从 $a$ 到 $n$（避免重复）
- 当 $a \times b$ 为偶数时，面积为整数
- 统计满足条件的组合数量

## 学习目标
- 掌握双重循环的使用
- 理解直角三角形的面积计算
- 学会去重（$a,b$ 和 $b,a$ 视为同一个）
