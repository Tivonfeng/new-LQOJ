# 乘法问题

## 🎯 问题描述
输入n个正整数，将它们相乘，若乘积超过1e6则输出">1000000"，否则输出乘积。

## 📝 任务要求
读取n个正整数，计算它们的乘积。如果乘积超过1000000，立即输出">1000000"；否则输出实际乘积。

## ⬇️ 输入格式
第一行正整数n（n≤50），后续n行各一个正整数（a≤100）。

## ⬆️ 输出格式
输出乘积或">1000000"。

## 📤 输出示例
```input1
3
2
3
5
```

```output1
15
```

```input2
3
100
100
100
```

```output2
1000000
```

```input3
4
100
100
100
2
```

```output3
>1000000
```

## 💡 学习提示
- 使用循环读取每个数字并累乘
- 在累乘过程中检查是否超过1000000
- 一旦超过就可以提前结束并输出结果
- 使用long long类型防止中间结果溢出

## 🎯 学习目标
- 掌握循环读取多个输入
- 理解累乘运算的应用场景
- 学会提前终止的优化策略
