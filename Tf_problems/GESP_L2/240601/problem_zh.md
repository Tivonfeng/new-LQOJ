# 平方之和

## 题目描述
给定 $n$ 个正整数，判断每个数是否能表示为两个正整数 $x$ 和 $y$ 的平方和（$x \times x + y \times y = a_i$）。

例如：
- $5 = 1^2 + 2^2 = 1 + 4$（可以表示）
- $4 = 2^2 + 0^2$，但 $0$ 不是正整数，所以不能表示

## 输入格式
第一行正整数 $n$（$1 \leq n \leq 10$），后续 $n$ 行各一个正整数（$1 \leq a_i \leq 10^6$）。

## 输出格式
每个数对应输出 "Yes" 或 "No"。

## 样例

```input1
2
5
4
```

```output1
Yes
No
```

### 样例解释
- $5 = 1^2 + 2^2$，可以表示为两个正整数的平方和
- $4$ 不能表示为两个正整数的平方和（$2^2 + 0^2$ 中的 $0$ 不是正整数）

## 数据范围
$1 \leq n \leq 10$，$1 \leq a_i \leq 10^6$

## 学习提示
- 遍历可能的 $x$ 值，计算 $y^2 = a_i - x^2$
- 检查 $y^2$ 是否为完全平方数
- 使用 `sqrt` 函数并检查是否为整数
- 注意 $x$ 和 $y$ 都必须是正整数

## 学习目标
- 掌握完全平方数的判断方法
- 理解数学问题的编程实现
- 学会优化循环减少计算量
