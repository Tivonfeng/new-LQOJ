### 编程题2：百鸡问题
- **问题描述**：扩展“百鸡问题”，每只公鸡X元，每只母鸡Y元，每Z只小鸡1元。现有n元买m只鸡，求方案数。
- **输入描述**：输入一行五个整数X、Y、Z、n、m（1≤X,Y,Z≤10，1≤n,m≤1000）。
- **输出描述**：输出方案数C。
- **样例输入1**：5 3 3 100 100
- **样例输出1**：4
- **样例输入2**：1 1 1 100 100
- **样例输出2**：5151
- **参考程序**：
```cpp
#include <iostream> 
using namespace std; 
int main() { 
    int x, y, z, n, m, cnt = 0; 
    cin >> x >> y >> z >> n >> m;
    for (int gj = 0; gj * x <= n && gj <= m; gj++) 
        for (int mj = 0; mj * y + gj * x <= n && mj + gj <= m; mj++) { 
            int xj = (n - gj * x - mj * y) * z; 
            if (gj + mj + xj == m) 
                cnt++;
        }
    cout << cnt << endl; 
    return 0; 
}
```

- **问题描述**：找出两个正整数A和B之间（包括A和B）的素数个数。素数是大于1的正整数，除了1和自身外不能被其他正整数整除。
- **输入描述**：输入2行，第一行正整数A，第二行正整数B（2≤A≤B≤1000）。
- **输出描述**：输出素数个数C。
- **样例输入1**：2 10
- **样例输出1**：4
- **样例输入2**：98 100
- **样例输出2**：0
- **参考程序**：
```cpp
#include <iostream>
using namespace std;
int main() {
    int a = 0, b = 0, cnt = 0;
    cin >> a >> b;
    for (int n = a; n <= b; n++) {
        bool isPrime = true;
        for (int i = 2; i * i <= n; i++) {
            if (n % i == 0) {
                isPrime = false;
                break;
            }
        }
        if (isPrime)
            cnt++;
    }
    cout << cnt << endl;
    return 0;
}
```

### 编程题2：自幂数判断
- **问题描述**：自幂数是指N位数的各位数字N次方之和等于本身。输入M个正整数，判断是否为自幂数。
- **输入描述**：第一行正整数M（1≤M≤100），后续M行各一个正整数（小于10⁸）。
- **输出描述**：每个数对应输出'T'（是）或'F'（否），可输入一个判断一个输出一个。
- **样例输入1**：3 152 111 153
- **样例输出1**：F F T
- **样例输入2**：5 8208 548834 88593477 12345 5432
- **样例输出2**：T T T F F
- **参考程序**：
```cpp
#include <iostream>
using namespace std; 
int main() {
    int m = 0;
    cin >> m; 
    for (int i = 0; i < m; i++) {
        int n = 0;
        cin >> n;
        int t = n, l = 0;
        while (t > 0) {
            t /= 10;
            l++;
        }
        int sum = 0;
        t = n;
        while (t > 0) {
            int d = t % 10;
            t /= 10;
            int mul = 1;
            for (int j = 0; j < l; j++)
                mul *= d;
            sum += mul;
        }
        if (sum == n)
            cout << "T" << endl;
        else
            cout << "F" << endl;
    }
    return 0;
}
```

```cpp
#include <stdio.h>
```

- **样例输出1**：Yes No
- **参考程序**：
```cpp
#include<bits/stdc++.h>
using namespace std;
bool check(int x) {
    int y = sqrt(x);
    return y * y == x;
}
int main() {
    int t;
    cin >> t;
    while (t--) {
        int n;
        cin >> n;
        int fl = 0;
        for (int i = 1; i * i < n; i++) {
            int j = n - i * i;
            if (check(j)) fl = 1;
        }
        if (fl) cout << "Yes\n";
        else cout << "No\n";
    }
    return 0;
}
```

```cpp
        while (b <= r) {
            n = a + b;
            if (n >= l && n <= r)
                cnt++;
            b *= 2;
        }
        a *= 2;
    }
    cout << cnt << endl;
    return 0;
}
```

```cpp
#include <algorithm>

int n, ans;
```

### 编程题2：黄金格
- **问题描述**：矩形地图H行W列，格子(r,c)满足√(r²+c²) ≤x+r-c时为黄金格，求黄金格数量。
- **输入描述**：输入3行，分别为H、W、x（正整数不超过100）。
- **输出描述**：输出黄金格数量。
- **样例输入1**：4 4 2
- **样例输出1**：4
- **参考程序**：
```cpp
#include <iostream>
#include <cmath>
using namespace std;
int main() {
    int H, W, x;
    cin >> H >> W >> x;
    int ans = 0;
    for (int r = 1; r <= H; ++r) {
        for (int c = 1; c <= W; ++c) {
            if (sqrt(r * r + c * c) <= x + r - c) {
                ans++;
            }
        }
    }
    cout << ans << "\n";
    return 0;
}
```
