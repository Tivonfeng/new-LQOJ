# 数位之和

## 🎯 问题描述
n个正整数中，判断每个数是否为"美丽数字"（每一位数字的总和是7的倍数）。

## 📝 任务要求
对于每个输入的正整数，计算其各位数字之和，判断是否为7的倍数。

## ⬇️ 输入格式
第一行正整数n（1≤n≤1e5），后续n行各一个正整数（1≤a_i≤1e5）。

## ⬆️ 输出格式
每个数对应输出"Yes"或"No"。

## 📤 输出示例
```input1
3
7
52
103
```

```output1
Yes
Yes
No
```

## 💡 学习提示
- 使用循环分解每个数的各位数字
- 累加各位数字得到总和
- 使用取模运算判断是否为7的倍数
- 注意处理大数据量的效率

## 🎯 学习目标
- 掌握数位分解的高效方法
- 理解取模运算的应用
- 学会处理大量数据的输入输出
