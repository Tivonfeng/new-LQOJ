# 黄金格

## 题目描述
矩形地图 $H$ 行 $W$ 列，格子 $(r, c)$ 满足 $\sqrt{r^2 + c^2} \leq x + r - c$ 时为黄金格，求黄金格数量。

## 输入格式
输入 3 行，分别为 $H$、$W$、$x$（正整数不超过 100）。

## 输出格式
输出黄金格数量。

## 样例

```input1
4
4
2
```

```output1
4
```

### 样例解释
对于 $H=4, W=4, x=2$，满足条件的格子有：
- $(1,1)$：$\sqrt{1^2+1^2} \approx 1.41 \leq 2 + 1 - 1 = 2$，是黄金格
- $(1,2)$：$\sqrt{1^2+2^2} \approx 2.24 \leq 2 + 1 - 2 = 1$，不是黄金格
- $(2,1)$：$\sqrt{2^2+1^2} \approx 2.24 \leq 2 + 2 - 1 = 3$，是黄金格
- $(2,2)$：$\sqrt{2^2+2^2} \approx 2.83 \leq 2 + 2 - 2 = 2$，不是黄金格
- $(3,1)$：$\sqrt{3^2+1^2} \approx 3.16 \leq 2 + 3 - 1 = 4$，是黄金格
- $(3,2)$：$\sqrt{3^2+2^2} \approx 3.61 \leq 2 + 3 - 2 = 3$，不是黄金格
- $(4,1)$：$\sqrt{4^2+1^2} \approx 4.12 \leq 2 + 4 - 1 = 5$，是黄金格
- $(4,2)$：$\sqrt{4^2+2^2} \approx 4.47 \leq 2 + 4 - 2 = 4$，是黄金格

共 4 个黄金格。

## 数据范围
$1 \leq H, W, x \leq 100$

## 学习提示
- 使用双重循环遍历所有格子 $(r, c)$
- 计算 $\sqrt{r^2 + c^2}$，可以使用 `sqrt` 函数
- 比较是否满足 $\sqrt{r^2 + c^2} \leq x + r - c$
- 注意数据类型的选择，避免浮点误差

## 学习目标
- 掌握双重循环的嵌套使用
- 理解平方根计算的应用
- 学会处理浮点数的比较
