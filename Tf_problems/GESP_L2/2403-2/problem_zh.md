# 小杨的日字矩阵

## 🎯 问题描述
构造N×N的日字矩阵（N为奇数），最左列、最右列是|，第一行、最后一行、中间一行（第(N+1)/2行）的第2~N-1个字符是-，其余为小写字母x。

## 📝 任务要求
根据输入的奇数N，输出N行N列的日字矩阵。

## ⬇️ 输入格式
一行整数N（5≤N≤49，奇数）。

## ⬆️ 输出格式
输出N行矩阵，每行N个字符，无额外符号。

## 📤 输出示例
```input1
5
```
```output1
|xxx|
|xxx|
|---|
|xxx|
|xxx|
```
```input2
7
```
```output2
|xxxxx|
|xxxxx|
|xxxxx|
|---|---|
|xxxxx|
|xxxxx|
|xxxxx|
```

## 💡 学习提示
- 使用双重循环遍历矩阵的每个位置
- 根据行列位置判断应该输出什么字符
- 关键行：第一行、最后一行、中间行
- 关键列：第一列、最后一列

## 🎯 学习目标
- 掌握二维数组/矩阵的遍历方法
- 理解多行多列的条件判断
- 学会处理多个边界条件
