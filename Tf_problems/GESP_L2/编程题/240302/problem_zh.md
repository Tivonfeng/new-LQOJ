# 小杨的日字矩阵

## 题目描述
构造 $N \times N$ 的日字矩阵（$N$ 为奇数），最左列、最右列是 `|`，第一行、最后一行、中间一行（第 $(N+1)/2$ 行）的第 2~$N-1$ 个字符是 `-`，其余为小写字母 `x`。

## 输入格式
一行整数 $N$（$5 \leq N \leq 49$，奇数）。

## 输出格式
输出 $N$ 行矩阵，每行 $N$ 个字符，无额外符号。

## 样例

```input1
5
```

```output1
|xxx|
|xxx|
|---|
|xxx|
|xxx|
```

```input2
7
```

```output2
|xxxxx|
|xxxxx|
|xxxxx|
|---|---|
|xxxxx|
|xxxxx|
|xxxxx|
```

## 数据范围
$5 \leq N \leq 49$，且 $N$ 为奇数

## 学习提示
- 使用双重循环遍历矩阵的每个位置
- 根据行列位置判断应该输出什么字符
- 关键行：第一行、最后一行、中间行
- 关键列：第一列、最后一列

## 学习目标
- 掌握二维数组/矩阵的遍历方法
- 理解多行多列的条件判断
- 学会处理多个边界条件
