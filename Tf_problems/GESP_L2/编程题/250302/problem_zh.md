# 时间跨越

## 题目描述
已知当前时间 $y$ 年 $m$ 月 $d$ 日 $h$ 时，计算 $k$ 小时后的时间 $y'$ 年 $m'$ 月 $d'$ 日 $h'$ 时。

## 输入格式
输入 5 行，分别为 $y$、$m$、$d$、$h$、$k$（$2000 \leq y \leq 3000$，$1 \leq m \leq 12$，$1 \leq d \leq 31$，$0 \leq h \leq 23$，$1 \leq k \leq 24$）。

## 输出格式
输出 $y'$、$m'$、$d'$、$h'$ 四个正整数。

## 样例

```input1
2008
2
28
23
1
```

```output1
2008 2 29 0
```

### 样例解释
2008 年是闰年，2 月有 29 天。当前时间是 2 月 28 日 23 时，加上 1 小时后变为 2 月 29 日 0 时。

## 数据范围
$2000 \leq y \leq 3000$，$1 \leq m \leq 12$，$1 \leq d \leq 31$，$0 \leq h \leq 23$，$1 \leq k \leq 24$

## 学习提示
- 先将小时相加，$h += k$
- 如果小时超过 24，需要进位到天数
- 处理不同月份的天数差异
- 注意闰年的 2 月有 29 天

## 学习目标
- 掌握日期时间的基本计算
- 理解闰年的判断方法
- 学会处理边界条件和进位
