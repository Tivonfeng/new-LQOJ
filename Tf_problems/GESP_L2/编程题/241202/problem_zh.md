# 数位和

## 题目描述
$n$ 个正整数中，求它们数位和的最大值（数位和是数字中所有数位的和）。

## 输入格式
第一行正整数 $n$（$1 \leq n \leq 10^5$），后续 $n$ 行各一个正整数（不超过 $10^{12}$）。

## 输出格式
输出数位和的最大值。

## 样例

```input1
3
16
81
10
```

```output1
9
```

### 样例解释
- $16$ 的数位和为 $1 + 6 = 7$
- $81$ 的数位和为 $8 + 1 = 9$
- $10$ 的数位和为 $1 + 0 = 1$

最大数位和为 $9$。

## 数据范围
$1 \leq n \leq 10^5$，每个数不超过 $10^{12}$

## 学习提示
- 数位和的计算：不断取余 10 并累加
- 使用循环遍历每个数字的所有位
- 使用 `long long` 类型存储大数字
- 实时维护最大值

## 学习目标
- 掌握数位和的计算方法
- 理解大数的处理方式
- 学会在遍历中维护极值
