# 找素数

## 题目描述
找出两个正整数 A 和 B 之间（包括 A 和 B）的素数个数。素数是大于 1 的正整数，除了 1 和自身外不能被其他正整数整除。

## 输入格式
2 行，第一行正整数 A，第二行正整数 B（$2 \leq A \leq B \leq 1000$）。

## 输出格式
输出素数个数 C。

## 样例

```input1
2
10
```

```output1
4
```

```input2
98
100
```

```output2
0
```

## 数据范围
$2 \leq A \leq B \leq 1000$

## 学习提示
- 使用循环遍历每个待检查的数
- 对每个数，检查从 2 到其平方根之间的所有整数是否能整除它
- 素数判断函数可以封装为独立函数

## 学习目标
- 掌握素数的判定方法
- 理解平方根优化原理
- 学会封装通用函数
