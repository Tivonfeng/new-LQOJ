# 优美的数字

## 题目描述
优美的数字是指十进制下所有数位都相同的正整数（如 $1, 2, \dots, 9, 11, 22, \dots, 99, 111, \dots$）。求不超过 $n$ 的正整数中有多少个优美的数字。

## 输入格式
输入一行正整数 $n$（$1 \leq n \leq 2025$）。

## 输出格式
输出优美的数字个数。

## 样例

```input1
6
```

```output1
6
```

### 样例解释
不超过 6 的优美数字有：$1, 2, 3, 4, 5, 6$，共 6 个。

```input2
2025
```

```output2
28
```

### 样例解释
优美的数字包括：
- 1 位数：$1 \sim 9$（9 个）
- 2 位数：$11, 22, \dots, 99$（9 个）
- 3 位数：$111, 222, \dots, 999$（9 个）
- 4 位数：$1111$（1 个，因为 $2222 > 2025$）

共计 $9 + 9 + 9 + 1 = 28$ 个。

## 数据范围
$1 \leq n \leq 2025$

## 学习提示
- 优美的数字可以是 1-9（单个数字）
- 也可以是 $11, 22, 33, \dots, 99$（两个相同数字）
- $111, 222, \dots, 999$（三个相同数字）
- 以此类推
- 可以用循环构造这些数字并判断是否不超过 $n$

## 学习目标
- 掌握数字位数的遍历方法
- 理解优美的数字的构成规律
- 学会统计符合条件的数字数量
