# 乘法问题

## 题目描述
输入 $n$ 个正整数，将它们相乘，若乘积超过 $10^6$ 则输出 ">1000000"，否则输出乘积。

## 输入格式
第一行正整数 $n$（$n \leq 50$），后续 $n$ 行各一个正整数（$a \leq 100$）。

## 输出格式
输出乘积或 ">1000000"。

## 样例

```input1
2
3
5
```

```output1
15
```

```input2
3
100
100
100
```

```output2
1000000
```

```input3
4
100
100
100
2
```

```output3
>1000000
```

## 数据范围
$n \leq 50$，$a \leq 100$

## 学习提示
- 使用循环读取每个数字并累乘
- 在累乘过程中检查是否超过 1000000
- 一旦超过就可以提前结束并输出结果
- 使用 `long long` 类型防止中间结果溢出

## 学习目标
- 掌握循环读取多个输入
- 理解累乘运算的应用场景
- 学会提前终止的优化策略
