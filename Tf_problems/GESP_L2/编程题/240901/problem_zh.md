# 数位之和

## 题目描述
$n$ 个正整数中，判断每个数是否为"美丽数字"（每一位数字的总和是 7 的倍数）。

## 输入格式
第一行正整数 $n$（$1 \leq n \leq 10^5$），后续 $n$ 行各一个正整数（$1 \leq a_i \leq 10^5$）。

## 输出格式
每个数对应输出 "Yes" 或 "No"。

## 样例

```input1
3
7
52
103
```

```output1
Yes
Yes
No
```

### 样例解释
- $7$ 的数位和为 $7$，$7 \div 7 = 1$ 余 $0$，是 7 的倍数
- $52$ 的数位和为 $5 + 2 = 7$，是 7 的倍数
- $103$ 的数位和为 $1 + 0 + 3 = 4$，不是 7 的倍数

## 数据范围
$1 \leq n \leq 10^5$，$1 \leq a_i \leq 10^5$

## 学习提示
- 使用循环分解每个数的各位数字
- 累加各位数字得到总和
- 使用取模运算判断是否为 7 的倍数
- 注意处理大数据量的效率

## 学习目标
- 掌握数位分解的高效方法
- 理解取模运算的应用
- 学会处理大量数据的输入输出
