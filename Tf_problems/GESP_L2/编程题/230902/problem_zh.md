# 小明的幸运数

## 题目描述
个位数为 k 的正整数及 k 的倍数称为"幸运数"，求 L 和 R 之间（包括 L 和 R）所有幸运数的和。

## 输入格式
输入 3 行，分别为 k（$2 \leq k \leq 9$）、L、R（$1 \leq L \leq R \leq 1000$）。

## 输出格式
输出幸运数之和。

## 样例

```input1
7
1
10
```

```output1
7
```

```input2
7
10
20
```

```output2
31
```

## 数据范围
$2 \leq k \leq 9$，$1 \leq L \leq R \leq 1000$

## 学习提示
- 遍历 L 到 R 之间的每个数
- 检查两个条件：个位数等于 k，或是 k 的倍数
- 满足条件的数累加到总和中

## 学习目标
- 掌握循环遍历的应用
- 理解取模运算判断倍数关系
- 学会组合多个判断条件
