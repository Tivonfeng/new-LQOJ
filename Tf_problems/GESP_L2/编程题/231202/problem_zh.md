# 小杨的 H 字矩阵

## 题目描述
构造 $N \times N$ 的 H 字矩阵（$N$ 为奇数），最左列、最右列是 `|`，中间一行（第 $(N+1)/2$ 行）的第 2~$N-1$ 个字符是 `-`，其余为小写字母 `a`。

## 输入格式
一行整数 $N$（$5 \leq N \leq 49$，奇数）。

## 输出格式
输出 $N$ 行矩阵，每行 $N$ 个字符，无额外符号。

## 样例

```input1
5
```

```output1
|aaa|
|aaa|
|---|
|aaa|
|aaa|
```

```input2
7
```

```output2
|aaaaa|
|aaaaa|
|aaaaa|
|---|---|
|aaaaa|
|aaaaa|
|aaaaa|
```

## 数据范围
$5 \leq N \leq 49$，且 $N$ 为奇数

## 学习提示
- 使用双重循环遍历矩阵的每个位置
- 根据行列位置判断应该输出什么字符
- 关键位置：第一列、最后一列、中间行、中间列
- 注意字符的边界条件处理

## 学习目标
- 掌握二维数组/矩阵的遍历方法
- 理解多条件判断的组合逻辑
- 学会处理边界条件
