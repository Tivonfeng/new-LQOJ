# 买文具

## 题目描述
签字笔 2 元一支、记事本 5 元一本、直尺 3 元一把。输入小明购买的数量及手中的钱，判断是否够买并输出剩余或缺少的钱。

## 输入格式
输入 4 行，分别为签字笔、记事本、直尺的数量（$1 \leq X,Y,Z \leq 10$）和手中的钱数。

## 输出格式
够买输出 "Yes" 和剩余钱数，否则输出 "No" 和缺少的钱数。

## 样例

```input1
1
1
1
20
```

```output1
Yes 10
```

```input2
1
1
1
5
```

```output2
No 5
```

## 学习提示
- 先计算总价：X*2 + Y*5 + Z*3
- 比较总价与手中的钱
- 使用条件判断选择输出内容

## 学习目标
- 掌握基本算术运算
- 理解条件判断的应用
- 学会处理多个输入的情况
