# 画三角形

## 题目描述
输入正整数 $n$，用大写字母拼成三角形图案。第 1 行 1 个字母，第 2 行 2 个字母……由上至下、由左至右依次用 A-Z 填充，Z 之后从头用 A 填充。

## 输入格式
一行正整数 $n$（$2 \leq n \leq 40$）。

## 输出格式
输出 $n$ 行三角形图案，每行字母数等于行号。

## 样例

```input1
3
```

```output1
A
BC
DEF
```

```input2
7
```

```output2
A
BC
DEF
GHIJ
KLMNO
PQRSTU
VWXYZAB
```

## 数据范围
$2 \leq n \leq 40$

## 学习提示
- 使用双重循环：外层循环控制行数，内层循环控制每行的字母数
- 利用 ASCII 码转换：'A' + offset 可以得到对应的字母
- 使用取模运算 %26 实现循环填充 A-Z

## 学习目标
- 掌握双重循环的嵌套使用
- 理解 ASCII 字符与整数的转换关系
- 学会使用取模运算实现循环效果
