# 菱形

## 题目描述
输入正整数 $n$，输出边长为 $n$ 的菱形字符画。菱形由 $2n-1$ 行组成，奇数行显示字符 `*`，偶数行显示字符 `#`。

## 输入格式
输入一行正整数 $n$（$1 \leq n \leq 10$）。

## 输出格式
输出 $2n-1$ 行菱形字符画。

## 样例

```input1
1
```

```output1
*
```

```input2
2
```

```output2
*
#
*
```

```input3
3
```

```output3
*
#
*
#
*
```

## 数据范围
$1 \leq n \leq 10$

## 学习提示
- 使用循环输出 $2n-1$ 行
- 奇数行输出 `*`，偶数行输出 `#`
- 可以使用取模运算判断奇偶性

## 学习目标
- 掌握循环控制
- 理解奇偶数的判断方法
- 学会生成规律性图案
