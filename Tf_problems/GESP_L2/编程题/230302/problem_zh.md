# 百鸡问题

## 题目描述
扩展"百鸡问题"，每只公鸡 X 元，每只母鸡 Y 元，每 Z 只小鸡 1 元。现有 n 元买 m 只鸡，求方案数。

## 输入格式
一行五个整数 X、Y、Z、n、m（$1 \leq X,Y,Z \leq 10$，$1 \leq n,m \leq 1000$）。

## 输出格式
输出方案数 C。

## 样例

```input1
5 3 3 100 100
```

```output1
4
```

```input2
1 1 1 100 100
```

```output2
5151
```

## 数据范围
$1 \leq X,Y,Z \leq 10$，$1 \leq n,m \leq 1000$

## 学习提示
- 使用三重循环遍历公鸡、母鸡、小鸡的数量
- 注意剪枝优化：检查当前数量是否超过金额或总数限制
- 小鸡数量 = (剩余金额) × Z

## 学习目标
- 掌握多重循环的嵌套使用
- 理解约束条件的检查方法
- 学会优化循环减少计算量
