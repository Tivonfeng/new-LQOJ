# 小杨做题

## 题目描述
第 1 天做 $a$ 道题，第 2 天做 $b$ 道题，从第 3 天起每天做题数是前两天总和。某一天做题数 $\geq m$ 则后续不做题，求第 $N$ 天总共做的题数。

## 输入格式
输入 4 行，分别为 $a$、$b$、$m$、$N$（$0 \leq a,b \leq 10$，$a,b < m \leq 10^6$，$3 \leq N \leq 364$）。

## 输出格式
输出总题数。

## 样例

```input1
1
2
10
5
```

```output1
19
```

### 样例解释
- 第 1 天：1 道题，累计 1
- 第 2 天：2 道题，累计 3
- 第 3 天：$1+2=3$ 道题，累计 6
- 第 4 天：$2+3=5$ 道题，累计 11
- 第 5 天：$3+5=8$ 道题，累计 19

```input2
1
1
5
8
```

```output2
12
```

### 样例解释
- 第 1 天：1 道题，累计 1
- 第 2 天：1 道题，累计 2
- 第 3 天：$1+1=2$ 道题，累计 4
- 第 4 天：$1+2=3$ 道题，累计 7
- 第 5 天：$2+3=5$ 道题（达到 $m=5$，后续不再做题）
- 第 6~8 天：不做题
- 总题数为 12

## 数据范围
$0 \leq a,b \leq 10$，$a,b < m \leq 10^6$，$3 \leq N \leq 364$

## 学习提示
- 使用循环计算每天的做题数
- 当某天做题数达到 $m$ 时停止
- 每天的做题数是前两天之和
- 使用 `long long` 防止溢出

## 学习目标
- 掌握循环控制流程
- 理解递推关系的代码实现
- 学会处理提前终止的情况
