# 小杨的 N 字矩阵

## 题目描述
构造 $n \times n$ 的 N 字矩阵（$n$ 为奇数），从左上角到右下角的对角线、第 1 列和第 $n$ 列都是 `+`，其余为 `-`。

## 输入格式
输入一行整数 $n$（$3 \leq n \leq 49$，奇数）。

## 输出格式
输出 $n$ 行 $n$ 列的矩阵。

## 样例

```input1
5
```

```output1
++--+
+-+-+
+---+
+-+-+
++--+
```

### 样例解释
N 字矩阵的特点：
- 第 1 列全是 `+`
- 第 $n$ 列全是 `+`
- 主对角线（行号等于列号）全是 `+`
- 其余位置是 `-`

## 数据范围
$3 \leq n \leq 49$，且 $n$ 为奇数

## 学习提示
- 使用双重循环遍历矩阵的每个位置
- 条件判断：第 1 列（$j=1$）或第 $n$ 列（$j=n$）或对角线（$i=j$）时输出 `+`
- 其他位置输出 `-`
- 学会用坐标判断来生成图案

## 学习目标
- 掌握双重循环的使用
- 理解二维矩阵的构造方法
- 学会用条件判断生成特定图案
