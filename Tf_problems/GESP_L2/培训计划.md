## 🔥 核心考点详解

### 1️⃣ 数位处理（**最高频考点**）

**题型特点**：分解整数的各位数字，进行求和、比较或统计

| 题目 | 核心操作 |
|-----|---------|
| 数位之和 | 循环取余/除以10求各位和 |
| 计数 | 统计1~n中数字k出现的次数 |
| 自幂数判断 | 分解位数 + 幂运算 |
| 数位和 | 大数（≤10¹²）的数位求和 |
| 优美的数字 | 判断所有位是否相同 |

**模板代码**：

```cpp
int sum = 0;
while (num > 0) {
    sum += num % 10;  // 取出个位
    num /= 10;         // 去掉个位
}
```

**易错点**：
- 0 的数位和是 0
- 负数需要特殊处理
- 大数要用 `long long`

---

### 2️⃣ 双重循环/矩阵构造（**必考题型**）

**题型特点**：遍历二维空间，根据位置输出不同字符

| 题目 | 输出规律 |
|-----|---------|
| H字矩阵 | 第一列、最后一列、中间行 |
| 日字矩阵 | 第一/最后/中间行 + 左右两列 |
| N字矩阵 | 第1列、第n列、主对角线 |
| 等差矩阵 | 第i行第j列 = i × j |
| 黄金格 | 双重循环 + 平方根计算 |

**模板代码**：

```cpp
for (int i = 0; i < n; i++) {
    for (int j = 0; j < n; j++) {
        if (条件判断) 输出特定字符
        else 输出默认字符
    }
}
```

**关键条件判断**：

```cpp
// 列判断
if (j == 0 || j == n-1)  // 第一列或最后一列

// 行判断  
if (i == 0 || i == n-1 || i == mid)  // 第一行、最后一行、中间行

// 对角线判断
if (i == j)  // 主对角线
```

---

### 3️⃣ 素数与质数判断（**高频考点**）

**题型特点**：判断一个数是否为素数

**模板代码**：

```cpp
bool isPrime(int n) {
    if (n < 2) return false;
    for (int i = 2; i * i <= n; i++) {
        if (n % i == 0) return false;
    }
    return true;
}
```

**优化**：只需检查到 √n，时间复杂度 O(√n)

---

### 4️⃣ 枚举/穷举（**思想方法**）

**题型特点**：通过遍历所有可能组合来寻找答案

| 题目 | 枚举策略 |
|-----|---------|
| 百鸡问题 | 三重循环遍历公鸡、母鸡数量 |
| 平方之和 | 遍历x，计算y² = a - x² |
| 数三角形 | 双重循环，a从1到n，b从a到n（去重） |
| 幂和数 | 预处理2的幂，两两求和 |

**优化技巧**：
- 减少循环范围（剪枝）
- 利用数学性质减少枚举量

---

### 5️⃣ 循环控制与提前终止

**题型特点**：循环过程中满足条件时提前结束

| 题目 | 场景 |
|-----|-----|
| 小杨做题 | 当做题数 ≥ m 时停止 |
| 乘法问题 | 乘积 > 10⁶ 时提前结束 |

**模板**：

```cpp
for (int i = 1; i <= N; i++) {
    // 计算当前值
    if (达到阈值) break;
    // 继续累加
}
```

---

### 6️⃣ 特殊数字判断

| 数字类型 | 定义 | 判断方法 |
|---------|-----|---------|
| 自幂数 | 各位数字位数次方之和 = 本身 | 数位分解 + 幂运算 |
| 幂和数 | n = 2^x + 2^y | 预处理幂次 + 双重循环 |
| 幸运数 | 个位数=k 或 k的倍数 | 取模判断 |
| 美丽数字 | 数位和是7的倍数 | 数位求和 + %7 |

---

### 7️⃣ 图案输出规律

**三角形**：

```cpp
// 第i行输出i个字符
for (int i = 0; i < n; i++) {
    for (int j = 0; j <= i; j++) {
        // 输出字符
    }
}
```

**菱形**：

```cpp
// 输出 2n-1 行
// 奇数行: *，偶数行: #
for (int i = 1; i <= 2*n-1; i++) {
    if (i % 2 == 1) cout << "*";
    else cout << "#";
}
```

bool isLeap(int year) {

```

**时间加法**：注意月份天数差异、闰年处理、进位

### 易错点提醒

| 易错点 | 示例 | 正确做法 |
|-------|-----|---------|
| 变量溢出 | 累乘用 int | 用 long long |
| 边界条件 | 素数判断漏掉 2 | 特殊处理 n < 2 |
| 循环范围 | 矩阵从0还是1开始 | 与输入/输出保持一致 |
| 去重 | (a,b) 和 (b,a) | 固定 a ≤ b |

int digitSum(int n) {
    int sum = 0;
    while (n > 0) {
        sum += n % 10;  // 取出最后一位
        n /= 10;         // 去掉最后一位
    }
    return sum;
}

// 方法2：处理0的情况
int digitSum(int n) {
    if (n == 0) return 0;
    int sum = 0;
    while (n > 0) {
        sum += n % 10;
        n /= 10;
    }
    return sum;
}
```
