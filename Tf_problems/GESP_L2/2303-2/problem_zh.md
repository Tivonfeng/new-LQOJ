# 百鸡问题

## 🎯 问题描述
扩展"百鸡问题"：每只公鸡X元，每只母鸡Y元，每Z只小鸡1元。现有n元买m只鸡，求共有多少种购买方案。

## 📝 任务要求
根据输入的参数，计算能够正好用n元购买m只鸡的方案数量。

## ⬇️ 输入格式
一行五个整数X、Y、Z、n、m（1≤X,Y,Z≤10，1≤n,m≤1000）。

## ⬆️ 输出格式
输出方案数C。

## 📤 输出示例
```input1
5 3 3 100 100
```

```output1
4
```

```input2
1 1 1 100 100
```

```output2
5151
```

## 💡 学习提示
- 使用三重循环遍历公鸡、母鸡、小鸡的数量
- 注意小鸡的数量必须是Z的倍数
- 使用break提前结束无效循环，提高效率

## 🎯 学习目标
- 掌握多重循环的应用
- 理解数学问题转化为编程逻辑的过程
- 学会优化循环减少计算量
