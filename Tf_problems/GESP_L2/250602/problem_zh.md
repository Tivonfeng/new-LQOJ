# 幂和数

## 题目描述
幂和数是指能表示为两个 2 的次幂之和的正整数（$n = 2^x + 2^y$，$x,y$ 为非负整数），求 $l$ 到 $r$ 之间的幂和数个数。

例如：
- $3 = 2^0 + 2^1 = 1 + 2$（是幂和数）
- $6 = 2^1 + 2^2 = 2 + 4$（是幂和数）
- $5 = 2^0 + 2^2 = 1 + 4$（是幂和数）

## 输入格式
输入一行两个正整数 $l$、$r$（$1 \leq l \leq r \leq 10^4$）。

## 输出格式
输出幂和数个数。

## 样例

```input1
2 8
```

```output1
6
```

### 样例解释
在 $2$ 到 $8$ 之间的幂和数有：$2, 3, 4, 5, 6, 8$，共 6 个。

```input2
10 100
```

```output2
20
```

## 数据范围
$1 \leq l \leq r \leq 10^4$

## 学习提示
- 预处理所有 2 的幂次（1, 2, 4, 8, 16, ...）
- 使用双重循环计算所有可能的和
- 统计落在 $[l, r]$ 区间内的和的数量
- 注意去重：相同的和只计算一次

## 学习目标
- 掌握双重循环的使用
- 理解幂次和的概念
- 学会区间统计的方法
