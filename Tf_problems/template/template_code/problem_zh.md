# 题目标题模板

## 🎯 问题描述

这里写问题的背景介绍和情境描述。可以使用游戏化、故事化的方式让题目更有趣，但要保持简洁明了。

## 📝 任务要求

明确说明学生需要完成什么任务，使用什么知识点或函数。

## ⬇️ 输入格式

描述输入的具体格式，如果没有输入则写"无输入"。

## ⬆️ 输出格式

描述输出的具体格式要求，包括格式、精度、换行等要求。

## 📤 输出示例

```input1
输入内容（如果没有输入则写"无"）
```

```output1
期望的输出内容
```

## 💡 学习提示

- 第一个技术要点或提示
- 第二个技术要点或提示  
- 第三个技术要点或提示

## 🎯 学习目标

- 掌握的第一个知识点
- 掌握的第二个知识点
- 掌握的第三个知识点
