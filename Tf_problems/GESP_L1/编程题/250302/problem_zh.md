# 四舍五入

## 题目描述
四舍五入是我们从小就熟悉的近似计算方法，现在需要你用代码实现一个“整十数四舍五入工具”！给定n个整数，你要把每个数精准四舍五入到最接近的整十数——比如43靠近40就舍为40，58靠近60就入为60，快来完成这个实用的小工具吧！


## 输入格式
共n+1行：
- 第一行：整数n（表示后续需要处理的整数个数，1≤n≤100）
- 接下来n行：每行一个整数a_i（需要四舍五入的整数，1≤a_i≤10000）

## 输出格式
共n行，每行一个整数，表示对应输入整数四舍五入到最接近整十数的结果。

## 样例
```input1
5
43
58
25
67
90
```

```output1
40
60
30
70
90
```
