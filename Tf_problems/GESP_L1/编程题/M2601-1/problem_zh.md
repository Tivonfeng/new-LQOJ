# 时钟角度计算

## 题目描述

小杨想知道给定的一个时刻（小时和分钟），时钟上时针和分针之间的夹角是多少度。

已知：
- 时钟上有 12 个大格，每个大格 30 度（360/12=30）
- 时针每小时走 30 度，每分钟走 0.5 度
- 分针每分钟走 6 度（360/60=6）

请你编写程序计算时针和分针之间的最小夹角（0~180度之间）。

## 输入格式

一行，两个整数 H 和 M，表示小时（1~12）和分钟（0~59）。

## 输出格式

一个整数，表示时针和分针之间的最小夹角（度数）。

## 样例
```input1
3 0
```

```output1
90
```

```input2
3 15
```

```output2
0
```

## 提示
注意：计算时针位置时，要加上分钟对时针的影响。
