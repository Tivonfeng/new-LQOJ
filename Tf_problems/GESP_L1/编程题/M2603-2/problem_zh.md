# 找出水仙花数

## 题目描述

小杨正在学习"水仙花数"。水仙花数是指一个 n 位数，它的每个位上的数字的 n 次幂之和等于它本身。

例如：153 是水仙花数，因为 1³ + 5³ + 3³ = 1 + 125 + 27 = 153。

请你编写程序，找出所有 m 到 n 之间的水仙花数。

## 输入格式

一行，两个整数 m 和 n（1 ≤ m ≤ n ≤ 10000）。

## 输出格式

每行一个水仙花数，按从小到大顺序输出。如果没有水仙花数，输出 "None"。

## 样例
```input1
100 999
```

```output1
153
370
371
407
```

```input2
1000 9999
```

```output2
1634
8208
9474
```

```input3
1 100
```

```output3
None
```

## 提示
水仙花数包括：
- 3位数：153, 370, 371, 407
- 4位数：1634, 8208, 9474
