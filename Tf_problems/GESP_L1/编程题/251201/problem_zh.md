# 手机电量显示
## 题目描述
小杨的手机有智能电量提醒功能！根据当前电量百分比P，会用不同方式提示：电量极低时显示“R”，电量偏低时显示“L”，电量充足时直接显示具体数值。请编写程序模拟这个提醒功能，处理多组电量数据的显示。

- 依次读取T组手机电量百分比P（正整数）；
- 根据P的范围输出对应结果：
  - P≤10：输出大写字母R；
  - 10<P≤20：输出大写字母L；
  - P>20：输出P的具体数值；
- 需使用多组输入处理、分支结构（if-else if-else）等知识点。

## 输入格式
第一行输入一个正整数T（1≤T≤20），代表数据组数；
接下来T行，每行输入一个正整数P（1≤P≤100），代表手机电量百分比。

## 输出格式
共T行输出，每组数据对应一行结果：
- 符合P≤10时输出“R”；
- 符合10<P≤20时输出“L”；
- 符合P>20时输出P的数值。

## 样例
```input1
5
10
1
20
99
19
```

```output1
R
R
L
99
L
```
