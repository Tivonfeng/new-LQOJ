# 找最大公约数

## 题目描述

小杨正在学习最大公约数（GCD）的概念。他想知道，对于给定的两个正整数 a 和 b，有多少个正整数能同时整除 a 和 b（即有多少个共同的因数）。

请你编写程序，计算两个正整数 a 和 b 的所有公因数的个数。

## 输入格式

一行，两个正整数 a 和 b（1 ≤ a, b ≤ 1000）。

## 输出格式

一个整数，表示 a 和 b 的公因数的个数。

## 样例
```input1
12 18
```

```output1
3
```

```input2
8 12
```

```output2
3
```

```input3
7 13
```

```output3
1
```

## 提示

12 和 18 的公因数有：1, 2, 3，共 3 个。
