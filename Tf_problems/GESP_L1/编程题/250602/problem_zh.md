# 值日

## 题目描述

小杨每 M 天值日一次，小红每 N 天值日一次。今天他们一起值日，求至少多少天后会再次同一天值日。

编写程序，读取两个正整数 M 和 N，输出再次同一天值日的最小天数（即 M 和 N 的最小公倍数）。

## 输入格式

输入包含两行，每行一个正整数：
- 第一行：正整数 M（小杨的值日周期）
- 第二行：正整数 N（小红的值日周期）

**数据范围**：M, N 均为正整数（可能很大，请使用 64 位整型）

## 输出格式

输出一行，一个整数，表示再次同一天值日的最小天数。

## 📤 样例1
```input1
4
6
```

```output1
12
```

## 样例解释
4 和 6 的最小公倍数是 12，所以 12 天后两人会再次同一天值日。
