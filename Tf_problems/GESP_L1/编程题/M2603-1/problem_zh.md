# 计算 BMI 值

## 题目描述

小杨想知道自己是否需要减肥。BMI（身体质量指数）的计算公式是：

**BMI = 体重(kg) / 身高(m)²**

请你根据输入的体重和身高，计算 BMI 值并判断体型：
- BMI < 18.5：体重过轻（输出 "under"）
- 18.5 ≤ BMI < 24：正常（输出 "normal"）
- 24 ≤ BMI < 28：过重（输出 "over"）
- BMI ≥ 28：肥胖（输出 "obese"）

## 输入格式

一行，两个整数 W 和 H，分别表示体重（kg）和身高（cm）。

## 输出格式

一行，一个字符串，表示体型分类。

## 样例
```input1
60 170
```

```output1
normal
```

```input2
80 170
```

```output2
over
```

```input3
50 180
```

```output3
under
```

## 提示
注意单位转换，身高需要从厘米转换为米。
