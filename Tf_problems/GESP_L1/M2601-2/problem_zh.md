# 找出完美数

## 题目描述

小杨正在学习"完美数"的概念：如果一个正整数的所有真因数（不包括它本身）的和等于它本身，那么这个数就是完美数。

例如：
- 6 的真因数有 1、2、3，它们的和是 6，所以 6 是完美数
- 28 的真因数有 1、2、4、7、14，它们的和是 28，所以 28 是完美数

请你编写程序，找出 1~n 之间所有的完美数并输出。

## 输入格式

一个整数 n（1 ≤ n ≤ 10000）。

## 输出格式

每行一个完美数，按从小到大顺序输出。如果没有完美数，输出 "None"。

## 样例
```input1
30
```

```output1
6
28
```

```input2
6
```

```output2
6
```

```input3
5
```

```output3
None
```

## 提示
找出每个数的所有真因数并求和，然后判断是否等于原数。
