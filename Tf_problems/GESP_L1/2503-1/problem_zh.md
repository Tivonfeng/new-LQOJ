# 图书馆里的老鼠

## 🎯 问题描述
图书馆里珍藏着n本好书，可一只贪吃的老鼠偷偷溜了进来！这只老鼠特别专注，每啃光一本书需要x小时，而且在啃完当前这本书之前，绝对不会切换到另一本。经过y小时的“疯狂啃书”，图书馆里还能剩下多少本完好无损的完整书籍呢？

## 📝 任务要求
根据输入的书籍总数n、老鼠啃完一本书的时间x、总经过时间y，通过整数运算和逻辑推导，计算并输出y小时后剩余的完整书籍数量。

## ⬇️ 输入格式
共3行，每行一个正整数：
- 第一行：书籍总数n
- 第二行：老鼠啃完一本书的时间x
- 第三行：总经过时间y

## ⬆️ 输出格式
一行一个整数，表示y小时后图书馆中剩余的完整书籍数量。

## 📤 输出示例
```input1
10
2
3
```

```output1
8
```

```input2
5
2
4
```

```output2
3
```

## 💡 学习提示
- 整数除法（y / x）可直接计算出老鼠在y小时内完整啃完的书籍数量
- 取余运算（y % x）能判断y小时时老鼠是否刚好啃完一本书（余数为0说明刚啃完，非0说明正在啃某本书）
- 剩余书籍数量的核心逻辑：总书数减去完整啃完的书数，再根据“是否正在啃书”调整（正在啃则该本书仍算完整）

## 🎯 学习目标
- 掌握C++中整数除法和取余运算的实际应用场景
- 学会将实际生活问题转化为数学逻辑和代码表达式
- 熟练掌握基础的输入输出函数（cin、cout）使用方法
