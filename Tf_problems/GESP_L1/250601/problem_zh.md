# 假期阅读

## 题目描述

小A有一本厚厚的书，总共有 N 页。小A一天最多能阅读 K 页，假期共有 T 天。求假期中最多能阅读完这本书的多少页。

编写程序，读取输入并输出假期中最多能阅读的页数（不超过书的总页数）。

## 输入格式

输入包含三行，每行是一个正整数：
- 第一行：正整数 N（书的总页数）
- 第二行：正整数 K（每天最多阅读页数）
- 第三行：正整数 T（假期天数）

**数据范围**：N, K, T 均不超过 1e9

## 输出格式

输出一行，一个整数，表示假期中最多能阅读的页数。

## 📤 样例1
```input1
19
3
6
```

```output1
18
```

## 样例解释
每天最多读 3 页，6 天最多读 3*6 = 18 页，书一共 19 页，所以最多读 18 页。
