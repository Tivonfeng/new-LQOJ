# 数字加密

## 题目描述

小杨发明了一种数字加密方法：对一个四位数进行如下操作：

1. 将每一位数字加 7
2. 将加 7 后的每一位数字对 10 取余
3. 交换千位和十位，交换百位和个位

例如：数字 1234
- 加7后：1→8, 2→9, 3→10→0, 4→11→1 → 8901
- 交换位置后：09 18 → 0981 即 981（注意：前导0不输出）

请你编写程序，对输入的四位数进行加密并输出结果。

## 输入格式

一个四位数 n（1000 ≤ n ≤ 9999）。

## 输出格式

加密后的数字（如果结果是0开头则不输出前导0）。

## 样例
```input1
1234
```

```output1
981
```

```input2
5555
```

```output2
2222
```

```input3
9999
```

```output3
6666
```
