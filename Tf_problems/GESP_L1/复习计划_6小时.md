#### 1️⃣ 循环结构专题（60分钟）

##### 核心概念
| 概念 | 说明 | 示例 |
|------|------|------|
| for循环 | 初始化→条件→循环体→增量 | `for(i=0;i<10;i++)` |
| while循环 | 先判断后执行 | `while(i<10){i++;}` |
| continue | 跳过后续，继续下一次 | `if(i%2==0) continue;` |
| break | 直接跳出循环 | `if(i==5) break;` |

##### 真题演练（4题）

**【题1】for循环边界**
> 代码执行后输出是（ ）。  
> `for(i=1;i<5;i++){if(i%3==0) break; printf("%d#",i);}`  
> A. 1#2#  B. 1#2#END  C. 1#2  D. 1#2#3#4#END  
> **答案：A** | 2509

**【题2】continue在for中**
> 代码执行后输出是（ ）。  
> `int n=0; for(int i=0;i<10;i++){if(i%2==0) continue; n+=i;}`  
> A. 55  B. 45  C. 25  D. 20  
> **答案：C** | 2503

**【题3】while循环边界**
> 代码执行后输出是（ ）。  
> `int num=0; while(num<=5){num+=1; if(num==3){printf("%d#",num); continue;} printf("%d#",num);}`  
> A. 1#2#4#5#6#  B. 1#2#4#5#6  C. 1#2#3#4#5#6#  D. 1#2#3#4#5#6  
> **答案：C** | 2509

**【题4】continue与break组合**
> 代码执行后输出是（ ）。  
> `int i; for(i=1;i<11;i+=3){continue; if(i%2==0) printf("%d#",i); break;} if(i>=11) printf("END");`  
> A. END  B. 1#  C. 1#4#END  D. 1#4#7#10#END  
> **答案：A** | 2506

##### 易错点提醒
- `i<=10` vs `i<10` 边界差异
- `continue` 会跳过 for 循环的增量部分
- `break` 只跳出当前循环（多层嵌套时需注意）

#### 2️⃣ 运算符与表达式（50分钟）

##### 核心公式
```
优先级：() > ++/-- > */% > +/- > >/>=//<=/< > ==/!= > ! > && > ||
```

##### 真题演练（5题）

**【题1】算术优先级**
> C++表达式 `14-3*3%2` 的值是（ ）。  
> A. 0  B. 11  C. 13  D. -67  
> **答案：C** | 解析：`14-3*3%2 = 14-(3*3)%2 = 14-9%2 = 14-1 = 13`

**【题2】取余运算**
> C++表达式 `3 * 4 % 5 / 6` 的值是（ ）。  
> A. 10  B. 5  C. 2  D. 0  
> **答案：D** | 解析：`3*4%5/6 = 12%5/6 = 2/6 = 0`

**【题3】逻辑非!**
> N是整型变量（值为5），表达式`(N + !N)`的值为（ ）。  
> A. 4  B. 5  C. 6  D. 0  
> **答案：B** | 解析：`!5 = 0`，`5+0=5`

**【题4】自增自减**
> `int a=16; ++a +=3;` 后a的值是（ ）。  
> A. 3  B. 17  C. 19  D. 20  
> **答案：D** | 解析：`++a = 17`，`17+3=20`

**【题5】连续比较陷阱**
> X是整型变量，表达式 `3<X<5` 的求值结果是（ ）。  
> A. 4  B. true  C. 1或0  D. 5  
> **答案：C** | 解析：`(3<X)<5` 结果为bool值

##### 易错点提醒
- `a*b` 无赋值则结果丢失
- `(++X)++` 编译错误
- `!0 = 1`, `!非0 = 0`
- `%` 运算符两边必须是整型

##### 命名规则
| 规则 | 正确示例 | 错误示例 |

> **答案：A** | 2512

**【题3】大小写敏感**

#### 5️⃣ 数值处理算法（50分钟）

##### 经典代码模板

```cpp
// 求逆序数
int n, new_number=0;
while(n!=0){
    new_number = new_number*10 + n%10;
    n /= 10;
}

// 交换两数
int temp = a; a = b; b = temp;
// 或
a = a - b; b = a + b; a = b - a;

// 求位数
int count = 0;
while(n > 0){
    count++;
    n /= 10;
}
```

##### 真题演练（4题）

**【题1】交换变量**
> 不能实现交换两个正整数a、b的代码是（ ）。  
> A. temp=a; a=b; b=temp  
> B. a=a-b; b=a+b; a=b-a  
> C. b=a-b; a=a+b; a=a-b  
> D. a,b=b,a  
> **答案：D** | 2509

**【题2】求镜面数**
> 求N的镜面数（如1234→4321），错误的循环条件是（ ）。  
> A. N != 0  B. not (N == 0)  C. N = 0  D. N > 0  
> **答案：C** | 2509

**【题3】求第M位数字**
> 求正整数N的第M位（个位为第1位），横线处应填入（ ）。  
> `int N,M,div=1; cin>>N>>M; for(i=0;i<M-1;i++)div*=10; cout<<(______);`  
> A. N%div/10  B. N/div/10  C. N%div%10  D. N/div%10  
> **答案：D** | 2509

**【题4】各位数字处理**
> 找出百位、十位、个位满足a²+b²=c²的三位数，横线处应填入（ ）。  
> `int a=i/100; int c=i%10; if(a*a+b*b==c*c) count++;`  
> A. b=(i/10)/10  B. b=(i/10)%10  C. b=(i%10)/10  D. b=(i%10)%10  
> **答案：B** | 2506

## 📝 考前必背速查表

### 运算符优先级（从高到低）
```
1. () 括号
2. ++ -- (前缀)
3. * / %
4. + -
5. < <= > >=
6. == !=
7. !
8. &&
9. ||
```

### 循环执行次数速查
```
for(i=0;i<10;i++)      → 10次
for(i=1;i<=10;i++)     → 10次
for(i=0;i<10;i+=2)     → 5次
while(i<10){i++}       → 10次
while(i<=10){i++}      → 10次
```

### 关键字不能用作变量名
```
int, float, double, char, bool
for, while, if, else, switch, case
return, break, continue
true, false, nullptr
```

### printf格式控制符
```
%d      整数
%02d    两位不足补0
%g      自动简化小数
%%      输出%
```
