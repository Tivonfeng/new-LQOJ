# 小杨的爱心快递
## 🎯 问题描述
小杨是爱心社区的志愿者，负责帮邻居寄送捐赠给山区小学的物资。快递公司为公益行动制定了特殊运费规则，既可以按体积计费，也可以按重量计费，最终会选择两种方式中价格更低的作为最终运费。快来帮小杨计算一下每件快递的实际运费吧！

## 📝 任务要求
- 读取快递的体积V、重量G、第一档重量运费M（G<300克时适用）、第二档重量运费N（G≥300克时适用）；
- 分别计算按体积计费（0.5×V）和按重量计费的金额；
- 输出两者中的较小值，结果保留1位小数；
- 需使用浮点数处理、条件判断（或三元运算符）、格式化输出等知识点。

## ⬇️ 输入格式
四行输入，每行一个一位小数的浮点数，依次为：
1. 快递体积V
2. 快递重量G
3. 第一档重量运费M
4. 第二档重量运费N

## ⬆️ 输出格式
一行输出，为实际快递运费，保留1位小数。

## 📤 输出示例
```input1
100.4
300.2
60.6
70.5
```

```output1
50.2
```

```input2
99.8
200.9
60.2
70.1
```

```output2
49.9
```

## 💡 学习提示
- 用scanf("%lf", &变量)读取浮点数（double类型），注意地址符&的使用；
- 按重量计费可通过三元运算符`(G < 300) ? M : N`快速判断；
- 比较两种计费方式的最小值，同样可使用三元运算符简化代码；
- 用printf("%.1f\n", result)实现保留1位小数的格式化输出。

## 🎯 学习目标
- 掌握double类型浮点数的输入、运算与输出；
- 熟练使用条件判断（三元运算符或if-else）解决二选一问题；
- 学会格式化输出指定精度的数值；
- 理解实际场景中“取较小值”的逻辑应用。
