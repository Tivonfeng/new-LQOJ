
| 规则 | 说明 |

## 四、运算符与表达式

### 4.1 算术运算符

| 运算符 | 说明 | 示例 | 结果 |
|--------|------|------|------|
| `+` | 加 | `10 + 5` | 15 |
| `-` | 减 | `10 - 5` | 5 |
| `*` | 乘 | `10 * 5` | 50 |
| `/` | 除 | `10 / 3` | 3（整除） |
| `%` | 取余 | `10 % 3` | 1 |

**重要规则**：
- 整数相除结果取整（向下取整）
- 取余运算符号与被除数相同

**经典真题**：

**【真题2312-2】** C++表达式10 - 3 * (2 + 1) % 10 的值是( )。
- A. 0
- B. 1
- C. 2
- D. 3

**答案**：B

**解析**：
- 先算括号：`2 + 1 = 3`
- 再算乘除取余：`3 * 3 % 10 = 9 % 10 = 9`
- 最后算加减：`10 - 9 = 1`

### 4.2 复合赋值运算符

| 运算符 | 等价于 | 示例 |
|--------|--------|------|
| `+=` | `a = a + b` | `a += 3` |
| `-=` | `a = a - b` | `a -= 3` |
| `*=` | `a = a * b` | `a *= 3` |
| `/=` | `a = a / b` | `a /= 3` |
| `%=` | `a = a % b` | `a %= 3` |

**经典真题**：

**【真题2303-10】** 如果a为int类型的变量，且a的值为6，则执行a *=3;之后，a的值会是( )。
- A. 3
- B. 6
- C. 9
- D. 18

**答案**：D

**解析**：`a *= 3` 等价于 `a = a * 3 = 6 * 3 = 18`

### 4.3 关系运算符

| 运算符 | 说明 | 示例 | 结果 |
|--------|------|------|------|
| `==` | 等于 | `5 == 3` | false |
| `!=` | 不等于 | `5 != 3` | true |
| `>` | 大于 | `5 > 3` | true |
| `<` | 小于 | `5 < 3` | false |
| `>=` | 大于等于 | `5 >= 3` | true |
| `<=` | 小于等于 | `5 <= 3` | false |

**经典真题**：

**【真题2303-11】** 如果a和b均为int类型的变量，下列表达式不能正确判断"a等于0且b等于0"的是( )
- A. `(a==0) && (b==0)`
- B. `(a==b==0)`
- C. `(!a) && (!b)`
- D. `(a==0)+(b==0)==2`

**答案**：B

**解析**：`a==b==0` 会被解释为 `(a==b) == 0`，结果恒为false。

### 4.4 逻辑运算符

| 运算符 | 说明 | 示例 | 结果 |
|--------|------|------|------|
| `&&` | 逻辑与 | `true && false` | false |
| `||` | 逻辑或 | `true || false` | true |
| `!` | 逻辑非 | `!true` | false |

**短路特性**：
- `a && b`：若a为false，则b不执行
- `a || b`：若a为true，则b不执行

### 4.5 运算符优先级

```cpp
// 从高到低
// 1. 一元运算符：!、++、--（右结合）
// 2. 算术运算符：*、/、%（高于+、-）
// 3. 关系运算符：<、<=、>、>=（高于==、!=）
// 4. 逻辑运算符：&&（高于||）
// 5. 赋值运算符：=、+=、-= 等（右结合）
```

**经典真题**：

**【真题2403-4】** C++表达式12 - 3 * 2 && 2 的值是( )。
- A. 0
- B. 1
- C. 6
- D. 9

**答案**：B

**解析**：
- 先算乘除：`3 * 2 = 6`
- 再算加减：`12 - 6 = 6`
- 最后算逻辑：`6 && 2` → `true` → `1`

## 五、顺序结构与分支结构

### 5.1 if语句

```cpp
// 单分支
if (条件) {
    语句块;
}

// 双分支
if (条件) {
    语句块1;
} else {
    语句块2;
}

// 多分支
if (条件1) {
    语句块1;
} else if (条件2) {
    语句块2;
} else {
    语句块3;
}
```

**经典真题**：

**【真题2309-7】** 下面C++代码执行后的输出是( )。

```cpp
int m=14;
int n=12;
if (m%2==0 && n%2==0)
    cout<<"都是偶数";
else if (m%2==1 && n%2==1)
    cout<<"都是奇数";
else
    cout<<"不都是偶数或奇数";
```

- A. 都是偶数
- B. 都是奇数
- C. 不都是偶数或奇数
- D. 以上说法都不正确

**答案**：A

**解析**：14是偶数，12也是偶数，进入第一个分支。

### 5.2 switch语句

```cpp
switch (表达式) {
    case 常量1:
        语句块1;
        break;
    case 常量2:
        语句块2;
        break;
    default:
        默认语句块;
        break;
}
```

**注意**：case后的break不能省略，否则会发生"穿透"。

**经典真题**：

**【真题2312-13】** 输入'C'时将输出的组号是( )。

```cpp
char c;
while (1){
    cin >> c;
    if(c=='q') break;
    switch(c){
        case 'A': cout <<"1 ";break;
        case 'B':cout <<"3";
        case 'C': cout <<"3";
        case 'D': cout <<"5";break;
        case 'E':cout <<"5";break;
        default: cout <<"9";
    }
}
```

- A. 3
- B. 3 5
- C. 3 5 9
- D. 以上都不对

**答案**：B

**解析**：输入'C'后，从case 'C'执行，输出"3"，然后执行到case 'D'也输出"5"。

## 六、循环结构

### 6.1 for循环

```cpp
for (初始化; 条件; 更新) {
    循环体;
}

// 示例
for (int i = 0; i < 10; i++) {
    cout << i;  // 输出0-9
}
```

**经典真题**：

**【真题2309-10】** 下面C++代码段执行后的输出是( )。

```cpp
int cnt=0;
for (int i=1;i<=5;i++)
    cnt=cnt+1;
cout << cnt;
```

- A. 1
- B. 4
- C. 5
- D. 10

**答案**：C

**解析**：循环执行5次（i=1,2,3,4,5），cnt每次加1，最终cnt=5。

### 6.2 while循环

```cpp
while (条件) {
    循环体;
}

// 示例
int i = 0;
while (i < 10) {
    cout << i;
    i++;
}
```

**经典真题**：

**【真题2309-12】** 下面C++代码执行后的输出是( )。

```cpp
int n=5;
int cnt=1;
while (n>=0){
    cnt+=1;
    n-=2;
}
cout << cnt;
```

- A. 3
- B. 4
- C. 6
- D. 7

**答案**：B

**解析**：n=5→3→1→-1，共执行4次（n=5,3,1,-1），cnt从1变为5。

### 6.3 do-while循环

```cpp
do {
    循环体;
} while (条件);
```

**特点**：循环体至少执行一次。

### 6.4 break与continue

| 语句 | 功能 |
|------|------|
| `break` | 跳出整个循环 |
| `continue` | 跳过本次循环剩余代码，进入下一次 |

**经典真题**：

**【真题2312-7】** 下面C++代码执行后的输出是( )。

```cpp
cnt=0;
for(int i=1;i<20;i++){
    if(i%2) continue;
    else if(i%3==0 && i%5==0) {
        cnt += i;
        break;
    }
}
cout << cnt;
```

- A. 90
- B. 44
- C. 20
- D. 10

**答案**：D

**解析**：i=15是第一个同时被2、3、5整除的数，累加后break跳出循环。

## 八、综合应用

### 8.1 质数判断

```cpp
// 方法1：统计因数个数
int N;
cin >> N;
int cnt = 0;
for (int i = 1; i <= N; i++) {
    if (N % i == 0) {
        cnt++;
    }
}
if (cnt == 2) {
    cout << N << "是质数";
} else {
    cout << N << "不是质数";
}

// 方法2：标记法（更高效）
int N;
cin >> N;
bool isPrime = true;
for (int i = 2; i < N; i++) {
    if (N % i == 0) {
        isPrime = false;
        break;
    }
}
```

```cpp
// 求和

    sum += i;
}
// sum = 55

// 条件求和
int sum = 0;
for (int i = 1; i <= 20; i++) {
    if (i % 3 == 0 || i % 5 == 0) {
        sum += i;
    }
}
// sum = 3+5+6+9+10+12+15+18+20 = 98
```
