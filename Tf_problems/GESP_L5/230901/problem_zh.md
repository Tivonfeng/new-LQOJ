# 因数分解

## 题目描述
每个正整数都可以分解成素数的乘积，例如 \(6=2×3\)、\(20=2^2×5\)。给定一个正整数 \(N\)，按要求输出它的因数分解式。

## 输入格式
输入第一行包含一个正整数 \(N\)，约定 \(2 ≤ N ≤ 10^{12}\)。

## 输出格式
输出一行因数分解式，按质因数由小到大排列，乘号用星号 `*` 表示且左右各空一格；仅当素数出现多次时，合并为指数形式（用上箭头 `^` 表示，左右不空格）。

## 样例

```input1
6
```

```output1
2 * 3
```

```input2
20
```

```output2
2^2 * 5
```

```input3
23
```

```output3
23
```

### 样例解释
- \(6 = 2 × 3\)，按从小到大顺序输出
- \(20 = 2^2 × 5\)，质因数 2 出现 2 次，用指数形式表示
- \(23\) 是质数，直接输出本身

## 数据范围
\(2 ≤ N ≤ 10^{12}\)

## 学习提示
- 使用循环从 2 开始尝试因数分解
- 每次找到一个质因数后，统计其出现次数
- 使用取模运算判断是否能被当前质数整除
- 最后如果 \(N > 1\)，说明剩余的是一个质数

## 学习目标
- 掌握素数因数分解的基本方法
- 理解指数形式的表示方法
- 学会处理大整数的因数分解
