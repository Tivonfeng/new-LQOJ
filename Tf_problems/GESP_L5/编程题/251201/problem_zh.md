# 数字移动

## 题目描述
序列包含 n 个正整数（n 为偶数），恰好有 n/2 对不同正整数（每对各出现两次）。每次操作可将第 i 个数字移动到任意位置，花费为该数字的体力。需使每对相同数字相邻，且每次花费不超过 X，求最小的 X。

## 输入格式
第一行正整数 n（偶数，\(1 ≤ n ≤ 10^5\)）；第二行 n 个正整数 \(A_i\)（\(1 ≤ A_i ≤ 10^5\)），保证至少需一次操作。

## 输出格式
输出最小的 X。

## 样例

```input1
6
2 1 2 1 3 3
```

```output1
2
```

## 数据范围
\(1 ≤ n ≤ 10^5\)，\(1 ≤ A_i ≤ 10^5\)

## 学习提示
- 二分查找答案 X
- 检查是否存在方案使最大移动花费不超过 X
- 贪心验证：优先移动体力大的数字

## 学习目标
- 掌握二分查找
- 理解移动操作的约束
- 学会处理配对问题
