# 烹饪问题

## 题目描述
有 N 种食材，第 i 种美味度为 \(a_i\)。两种食材的契合度为其美味度的按位与运算（&）结果。找到契合度最高的两种食材，输出该契合度。

## 输入格式
第一行整数 N；第二行 N 个整数 \(a_0, ..., a_{N-1}\)（\(0 ≤ a_i ≤ 2^{31}-1\)）。

## 输出格式
输出一行整数，为最高契合度。

## 样例

```input1
3
1 2 3
```

```output1
2
```

```input2
5
5 6 2 10 13
```

```output2
8
```

### 样例解释
- 1&2=0，1&3=1，2&3=2，最高为 2
- 5&6=4，5&2=0，5&10=0，5&13=5，6&2=2，6&10=2，6&13=4，2&10=2，2&13=0，10&13=8，最高为 8

## 数据范围
\(1 ≤ N ≤ 10^6\)，\(0 ≤ a_i ≤ 2^{31}-1\)

## 学习提示
- 使用贪心 + 位运算的技巧
- 从高位到低位依次确定答案的每一位
- 使用快速排序或 partition 将数组按当前位分组
- 如果某一位后还有足够的数，则该位可以为 1

## 学习目标
- 掌握高位贪心的技巧
- 理解位运算的应用
- 学会处理大规模数据的位运算问题
