# 巧夺大奖

## 题目描述
小明参加游戏，有 n 个时间段和 n 个小游戏。每个小游戏有完成期限 \(T_i\)（需在第 \(T_i\) 个时间段结束前完成）和奖励 \(R_i\)，每个小游戏可在一个时间段内完成。安排每个时间段选择的小游戏，使总奖励最高。

## 输入格式
第一行含正整数 n（\(1 ≤ n ≤ 500\)）；第二行含 n 个正整数 \(T_i\)（\(1 ≤ T_i ≤ n\)）；第三行含 n 个正整数 \(R_i\)（\(1 ≤ R_i ≤ 1000\)）。

## 输出格式
输出一行整数 C，为最高可获得的奖励。

## 样例

```input1
7
4 2 4 3 1 4 6
70 60 50 40 30 20 10
```

```output1
230
```

### 样例解释
最优安排方式：
- 第1个时间段：完成第5个小游戏（T=1, R=30）
- 第2个时间段：完成第2个小游戏（T=2, R=60）
- 第3个时间段：完成第4个小游戏（T=3, R=40）
- 第4个时间段：完成第1个小游戏（T=4, R=70）
- 第5个时间段：完成第3个小游戏（T=4, R=50）
- 第6个时间段：完成第6个小游戏（T=4, R=20）
- 第7个时间段：完成第7个小游戏（T=6, R=10）

总奖励：30 + 60 + 40 + 70 + 50 + 20 + 10 = 230

## 数据范围
\(1 ≤ n ≤ 500\)，\(1 ≤ T_i ≤ n\)，\(1 ≤ R_i ≤ 1000\)

## 学习提示
- 使用贪心算法：优先选择奖励最高的小游戏
- 将所有小游戏按奖励从高到低排序
- 对于每个小游戏，尽量安排在它截止日期之前的最后一个空闲时间段
- 使用数组记录每个时间段是否已被占用

## 学习目标
- 掌握贪心算法的设计思想
- 理解任务调度的最优策略
- 学会使用数组模拟时间安排
