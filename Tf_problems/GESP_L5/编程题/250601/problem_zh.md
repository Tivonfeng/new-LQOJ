# 奖品兑换

## 题目描述
小A有 n 张课堂优秀券和 m 张作业优秀券，可通过两种方式兑换奖品：
1. a 张课堂券 + b 张作业券
2. b 张课堂券 + a 张作业券

求最多可兑换的奖品份数。

## 输入格式
第一行两个正整数 n、m（\(1 ≤ n, m ≤ 10^9\)）；第二行两个正整数 a、b（\(1 ≤ a, b ≤ 10^4\)）。

## 输出格式
输出最多可兑换的奖品份数。

## 样例

```input1
8 8
2 1
```

```output1
5
```

### 样例解释
- 方式1：2 课堂 + 1 作业
- 方式2：1 课堂 + 2 作业
- 最多可兑换 5 份

## 数据范围
\(1 ≤ n, m ≤ 10^9\)，\(1 ≤ a, b ≤ 10^4\)

## 学习提示
- 二分查找答案 v
- 检查 v 份奖品是否可行
- 贪心计算所需的课堂券和作业券数量

## 学习目标
- 掌握二分查找
- 理解兑换约束
- 学会处理优化问题
