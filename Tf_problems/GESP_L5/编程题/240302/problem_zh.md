# B-smooth数

## 题目描述
B-smooth数是指最大质因子不超过 B 的正整数。给定 n 和 B，求不超过 n 的 B-smooth数的数量。

## 输入格式
第一行两个正整数 n、B（\(1 ≤ n, B ≤ 10^6\)）。

## 输出格式
输出非负整数，为不超过 n 的 B-smooth数的数量。

## 样例

```input1
10 3
```

```output1
7
```

### 样例解释
B=3 时，最大质因子不超过 3 的数有：
1, 2, 3, 4, 6, 8, 9（共 7 个）

## 数据范围
\(1 ≤ n, B ≤ 10^6\)

## 学习提示
- 使用线性筛法计算每个数的最大质因子
- 在筛的过程中记录每个合数的最大质因子
- 最后统计最大质因子 ≤ B 的数的个数

## 学习目标
- 掌握线性筛法
- 理解 B-smooth 数的定义
- 学会处理质因子相关问题
