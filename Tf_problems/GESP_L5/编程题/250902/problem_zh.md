# 有趣的数字和

## 题目描述
二进制表示包含奇数个 1 的正整数是"有趣的"。给定 l、r，统计 l≤n≤r 的有趣整数之和。

## 输入格式
一行两个正整数 l、r（\(1 ≤ l ≤ r ≤ 10^9\)）。

## 输出格式
输出有趣整数的和（使用 long long）。

## 样例

```input1
3 8
```

```output1
19
```

### 样例解释
有趣的数：3(11), 5(101), 6(110), 9(1001), 10(1010), 12(1100)
3+5+6+9+10+12 = 45，题目 l=3,r=8，所以 3+5+6=14

修正：3~8 范围内有趣的数是 3(11), 5(101), 6(110)，和为 14

实际 1~8 范围内有趣的数：1(1), 3(11), 5(101), 6(110)
但题目范围是 3~8，所以是 3+5+6=14

等等，让我重新看样例输出是 19

实际上有趣的数在 3~8 范围：3(11), 5(101), 6(110), 7(111)
3+5+6+7 = 21，这也不对

让我再想想：3(11 有 2 个 1，是偶数，不是有趣的)
5(101 有 2 个 1，是偶数)
...

实际上：
1(1) - 1 个 1 - 奇数 - 有趣
2(10) - 1 个 1 - 奇数 - 有趣
3(11) - 2 个 1 - 偶数 - 无趣
4(100) - 1 个 1 - 奇数 - 有趣
5(101) - 2 个 1 - 偶数 - 无趣
6(110) - 2 个 1 - 偶数 - 无趣
7(111) - 3 个 1 - 奇数 - 有趣
8(1000) - 1 个 1 - 奇数 - 有趣

所以 3~8 范围：4, 7, 8
4+7+8 = 19，正确！

## 数据范围
\(1 ≤ l ≤ r ≤ 10^9\)

## 学习提示
- 使用数位 DP 或数学公式计算
- 递归处理每一位
- 区分当前位的奇偶性
- 统计满足条件的数及其和

## 学习目标
- 掌握数位 DP
- 理解二进制中 1 的个数奇偶性
- 学会处理区间统计问题
