# 最大公因数

## 题目描述
k≥3 个正整数的最大公因数定义为 \(gcd(c_1,c_2,...,c_k)=gcd(gcd(c_1,...,c_{k-1}),c_k)\)。给定 n 个正整数和 q 组询问，每组询问求 \(gcd(a_1+i,a_2+i,...,a_n+i)\)（i 为询问序号，1≤i≤q）。

## 输入格式
第一行两个正整数 n、q（\(1 ≤ n, q ≤ 10^5\)）；第二行 n 个正整数 \(a_1,...,a_n\)（\(1 ≤ a_i ≤ 1000\)）。

## 输出格式
q 行，每行输出对应询问的最大公因数。

## 样例

```input1
5 3
6 9 12 18 30
```

```output1
1
1
3
```

### 样例解释
- i=1：gcd(7,10,13,19,31) = 1
- i=2：gcd(8,11,14,20,32) = 1
- i=3：gcd(9,12,15,21,33) = 3

## 数据范围
\(1 ≤ n, q ≤ 10^5\)，\(1 ≤ a_i ≤ 1000\)

## 学习提示
- 先排序数组
- 计算相邻元素的差值
- gcd(a_1+i, ..., a_n+i) = gcd(gcd(所有差值), a_1+i)

## 学习目标
- 掌握 gcd 的性质
- 理解差分的应用
- 学会处理序列 gcd 查询问题
