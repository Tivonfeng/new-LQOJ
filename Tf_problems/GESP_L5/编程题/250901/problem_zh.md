# 数字选取

## 题目描述
从 1~n 的整数中选取若干整数，使任意两个不同整数互质（最大公因数为 1），最大化选取数量。

## 输入格式
一行正整数 n（\(1 ≤ n ≤ 10^5\)）。

## 输出格式
输出最大选取数量。

## 样例

```input1
6
```

```output1
4
```

```input2
9
```

```output2
5
```

### 样例解释
- n=6 时，选取 {1, 2, 3, 5}，共 4 个
- n=9 时，选取 {1, 2, 3, 5, 7}，共 5 个

## 数据范围
\(1 ≤ n ≤ 10^5\)

## 学习提示
- 选取所有不超过 n 的质数，再加上 1
- 1 与任何数互质
- 质数之间互质
- 答案为质数个数 + 1

## 学习目标
- 理解互质的概念
- 掌握素数筛法
- 学会构造最大互质集合
