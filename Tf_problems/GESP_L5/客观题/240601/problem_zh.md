# 黑白格

## 题目描述
n 行 m 列的网格图，每个格子为白色（0）或黑色（1）。求至少包含 k 个黑色格子的最小子矩形的格子数量，不存在则输出 0。

## 输入格式
第一行三个正整数 n、m、k（\(1 ≤ n, m ≤ 100\)，\(1 ≤ k ≤ n×m\)）；之后 n 行，每行一个长度为 m 的 01 串。

## 输出格式
输出满足条件的最小子矩形的格子数量，不存在则输出 0。

## 样例

```input1
4 5 5
00000
01111
00011
00011
```

```output1
6
```

## 数据范围
\(1 ≤ n, m ≤ 100\)，\(1 ≤ k ≤ n×m\)

## 学习提示
- 使用二维前缀和计算子矩形的黑色格子数量
- 枚举所有可能的子矩形
- 使用二分查找优化寻找最小高度
- 计算每个子矩形的面积

## 学习目标
- 掌握二维前缀和
- 理解子矩形枚举技巧
- 学会处理二维网格问题
