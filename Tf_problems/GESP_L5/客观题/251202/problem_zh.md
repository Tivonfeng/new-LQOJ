# 相等序列

## 题目描述
序列 A 有 N 个正整数，可执行两种操作（每次花费 1 金币）：
1. 将 \(A_i\) 变为 \(A_i×P\)（P 为任意质数）
2. 将 \(A_i\) 变为 \(A_i/P\)（P 为任意质数，需 \(A_i\) 能整除 P）

使序列所有整数相同，求最少花费。

## 输入格式
第一行正整数 N（\(1 ≤ N ≤ 10^5\)）；第二行 N 个正整数 \(A_i\)（\(1 ≤ A_i ≤ 10^5\)）。

## 输出格式
输出最少花费。

## 样例

```input1
5
10 6 35 105 42
```

```output1
8
```

## 数据范围
\(1 ≤ N ≤ 10^5\)，\(1 ≤ A_i ≤ 10^5\)

## 学习提示
- 将每个数分解为质因数及其指数
- 对每个质因子，找到中位数指数
- 计算每个数变为中位数所需的花费
- 求和得到总花费

## 学习目标
- 掌握质因数分解
- 理解中位数在优化中的作用
- 学会处理序列转换问题
