# 平均分配

## 题目描述
小A有 2n 件物品，小B和小C各买 n 件。第 i 件物品小B出价 \(b_i\)，小C出价 \(c_i\)。求小A卖出所有物品的最大收入。

## 输入格式
第一行正整数 n（\(1 ≤ n ≤ 10^5\)）；第二行 2n 个整数 \(b_1,...,b_{2n}\)（\(0 ≤ b_i ≤ 10^9\)）；第三行 2n 个整数 \(c_1,...,c_{2n}\)（\(0 ≤ c_i ≤ 10^9\)）。

## 输出格式
输出最大收入。

## 样例

```input1
1
3 5
4 6
```

```output1
9
```

### 样例解释
两件物品，一件给 B（出价 3），一件给 C（出价 6），总收入 9

## 数据范围
\(1 ≤ n ≤ 10^5\)，\(0 ≤ b_i, c_i ≤ 10^9\)

## 学习提示
- 初始假设所有物品都给 B
- 计算每个物品给 C 的额外收益：d_i = c_i - b_i
- 对 d_i 排序，选择最大的 n 个加到答案中

## 学习目标
- 掌握差值排序
- 理解最优分配策略
- 学会处理分配问题
