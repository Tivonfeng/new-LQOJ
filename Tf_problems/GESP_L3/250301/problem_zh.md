# 寻找最小y

## 题目描述
小 A 有一个整数 x，他想找到最小的正整数 y 使得下式成立：(x and y) + (x or y) = 2025。其中 and 表示二进制按位与运算，or 表示二进制按位或运算。如果不存在满足条件的 y，则输出-1。

## 输入格式
一行，一个整数 x。

## 输出格式
一行，一个整数，若满足条件的 y 存在则输出 y，否则输出-1。

## 样例

```input1
1025
```

```output1
1000
```

## 数据范围
0 ≤ x < 2025

## 学习提示
- 遍历 y 从 1 到 2025
- 计算 (x & y) + (x | y)
- 如果结果等于 2025，记录并输出

## 学习目标
- 掌握位运算的基本操作
- 理解按位与和按位或的关系
