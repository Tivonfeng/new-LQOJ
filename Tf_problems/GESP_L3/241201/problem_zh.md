# 数字替换

## 题目描述
小杨有一个包含 n 个数字的序列 A，即 A={a₁, a₂, ..., aₙ}，他想将其中大于 k 的数字都替换为序列的最大值，将其中小于 k 的数字都替换为序列的最小值，请你帮他计算出替换后的序列。

## 输入格式
第一行包含两个正整数 n、k，含义如题面所示。第二行包含 n 个数字，代表序列 A。

## 输出格式
输出 n 个整数，代表替换后的结果。

## 样例

```input1
10 2
-2 -2 0 2 2 1 5 0 -2 -1
```

```output1
-2 -2 -2 2 2 -2 5 -2 -2 -2
```

## 数据范围
1 ≤ n ≤ 10^5
|k|,|a_i| ≤ 10^5

## 学习提示
- 首先遍历数组找到最大值和最小值
- 再次遍历数组，根据条件进行替换
- 大于 k → 最大值，小于 k → 最小值，等于 k → 保持不变

## 学习目标
- 掌握数组的多次遍历
- 理解条件替换的逻辑
