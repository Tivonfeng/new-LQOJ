1. 导入与情境创设 (10分钟)

【故事导入】 大家好！想象一下，你遇到了一位来自代码星球的机器人朋友🤖，它只会说一种特殊的语言。你想让它帮你做一些神奇的事情，比如画画、计算或者控制一个游戏角色，你该怎么办呢？没错，你需要学习它的语言！今天我们要学习的C++，就是这样一种能和计算机这位超级助手沟通的"魔法语言"！

【展示样例】(教师可以快速演示一个简单的C++程序输出结果，例如输出一个有趣的图案或计算结果)

#include <bits/stdc++.h>
using namespace std;

int main() {
    cout << "********************" << endl;
    cout << "*  欢迎来到C++魔法世界! *" << endl;
    cout << "********************" << endl;
    cout << "1 + 1 = " << 1 + 1 << endl; 
    return 0;
}

看，只需要几行简单的"咒语"，计算机就能按照我们的想法输出文字和计算结果！

【明确目标】 这节课，我们的目标就是揭开C++的神秘面纱，学会编写并运行我们自己的第一个C++小程序，并理解其中的基本"魔法规则"。准备好成为一名C++小魔法师了吗？✨

2. 核心知识讲解 (20分钟)

🤔 到底什么是C++？

C++就像是一本魔法咒语书📜，通过它，你可以指挥计算机为你做各种事情：

你可以用C++做什么？

  • 🎮 创造酷炫的游戏
  • 🧮 进行快速复杂的计算
  • 🤖 控制机器人完成任务
  • �� 设计智能汽车的"大脑"
  • ... 还有更多！你每天用的微信、玩的很多游戏，背后都有C++的影子！

👋 你的第一个C++程序：向世界问好！

每个C++程序（魔法咒语）都需要一个固定的格式，让我们来看看最经典的"Hello World"程序：

#include <bits/stdc++.h> // 1. 引入魔法工具箱
using namespace std;    // 2. 使用标准工具

int main() { // 3. 魔法开始的地方 (主函数)
    // 4. 这是具体的魔法指令：让计算机说出...
    cout << "你好，世界！我是C++小魔法师！" << endl; // 5. 输出文字并换行

    return 0; // 6. 告诉计算机魔法成功结束
} // 魔法结束

【分步讲解】

  1. #include <bits/stdc++.h>: 像是在说："把所有常用的魔法工具（函数、功能）都给我准备好！" 这是一个"万能头文件"，包含了我们初学阶段几乎所有需要的功能。
  2. using namespace std;: 告诉计算机，我们要使用标准（std）的工具，这样后面用cout等工具时就不用每次都写全名std::cout了，更方便！
  3. int main() { ... }: 这是程序的"心脏"或者说"主角"——main函数。程序从这里开始执行，所有的主要指令都写在大括号{}里面。int表示这个函数执行完后会返回一个整数类型的结果。
  4. // ...: 这两个斜杠后面的文字叫做注释。它们是写给我们自己看的笔记，计算机在执行程序时会完全忽略它们。用来解释代码非常有用！
  5. cout << "你好，世界！我是C++小魔法师！" << endl;: 这是本程序的核心指令。
    • cout: 可以想象成计算机的"嘴巴"或"屏幕输出设备"。
    • <<: 是输出运算符，像一个箭头，把右边的内容"推"给cout去显示。
    • "...": 双引号里的内容是要输出的文字（字符串）。
    • endl: 表示输出结束后，换到下一行（end line）。
    • 注意：C++的大部分语句（指令）后面都需要加上英文分号;，表示这句话结束了。
  6. return 0;: 表示程序顺利执行完毕，并返回一个值0。通常，返回0代表"成功"。

快速检查

一个基本的C++程序必须包含哪个部分？ A. #include <iostream> (虽然常用，但不是绝对必须，可以用别的) B. using namespace std; (可以不写，但代码会变麻烦) C. int main() { ... } (程序入口，必须有！) D. cout << ...; (如果程序不需要输出，可以没有)

(思考一下...答案是C)

🧮 C++的简单计算：加法与减法

C++不仅能输出文字，还能像计算器一样进行计算！

#include <bits/stdc++.h>
using namespace std;

int main() {
    // 加法魔法：合并两堆宝石
    cout << "5 + 3 = "; // 先输出文字 "5 + 3 = "
    cout << 5 + 3 << endl; // 再计算 5+3 的结果并输出，然后换行

    // 减法魔法：分享糖果后剩下的
    cout << "10 - 4 = " << 10 - 4 << endl; // 直接在一行里输出文字和计算结果

    return 0;
}

【运行结果】

5 + 3 = 8
10 - 4 = 6

【讲解要点】

  • cout可以直接输出数字和计算表达式的结果。
  • 注意区分 "5 + 3" （这是文字）和 5 + 3 （这是计算）。
  • 可以使用 + 进行加法，- 进行减法。

3. 分层实践任务 (40分钟)

现在，轮到你来施展魔法了！请打开你的编程环境（例如Dev-C++、在线编译器等）。

阶段一：单人基础练习 (15分钟)

任务1：修改问候语 (🌟)

  • 目标：熟悉cout输出文字。
  • 要求：复制上面的"Hello World"代码，修改双引号里的内容，让程序输出你自己的名字和一句问候语。例如："你好，我是小明，很高兴学习C++！"
  • 代码锦囊（遇到困难时可以看）：
    • 提示1：找到cout << "..." << endl;这一行。
    • 提示2：只需要修改双引号""内部的文字。
    • 提示3：确保代码的其他部分（如#include, main, {} , ;）都正确无误。

任务2：简单的加减法 (🌟)

  • 目标：练习使用cout输出计算结果。
  • 要求：编写一个新的C++程序（或者在刚才的代码return 0;之前添加），计算并输出以下两个算式的结果：
    • 15 + 8
    • 20 - 7
    • 输出格式要求：每行一个算式及其结果，例如：
      15 + 8 = 23
20 - 7 = 13
  • 代码锦囊：
    • 提示1：你需要使用cout << ... << endl; 两次。
    • 提示2：一次输出计算 15 + 8 的结果，一次输出 20 - 7 的结果。
    • 提示3：可以参考讲解中的计算示例，结合输出文字和计算表达式。

阶段二：进阶挑战/小组合作 (25分钟)

（可以选择个人继续挑战，或与旁边同学组成临时小组讨论完成）

任务3：添加注释 (🌟🌟)

  • 目标：学会在代码中添加注释。
  • 要求：在你完成的任务2代码中，为每一行cout语句添加单行注释//，说明这行代码的作用。例如：
    cout << "15 + 8 = " << 15 + 8 << endl; // 计算并输出15加8的结果
cout << "20 - 7 = " << 20 - 7 << endl; // 计算并输出20减7的结果
  • 思考：为什么写注释很重要？

任务4：混合输出 (🌟🌟)

  • 目标：练习在一行cout中混合输出文字和多个计算结果。
  • 要求：编写代码，只用一行cout语句，输出类似下面的内容（具体数字不要求完全一样，但格式要类似）：
    我的幸运数字是：7，我今年的目标是跑 100 公里！
    其中 7 和 100 需要是直接写在代码里的数字，而不是写在双引号里。
  • 代码锦囊：
    • 提示1：你可以连续使用多个 << 输出运算符。
    • 提示2：cout << "文字1" << 数字1 << "文字2" << 数字2 << "文字3" << endl;

任务5：小小计算器 (🌟🌟🌟 - 挑战！)

  • 目标：综合运用本节课知识。
  • 要求：创建一个程序，计算：
    1. 你的年龄加上5岁是多少？
    2. 100减去你的幸运数字（假设是7）是多少？
  • 输出格式：
    我5年后将是 XX 岁。
100减去我的幸运数字7是 YY。
    其中XX和YY需要是计算出的结果。
  • 代码锦囊：
    • 提示1：你需要定义两个计算，可以用 你的年龄 + 5 和 100 - 7 这样的表达式。
    • 提示2：使用两个cout语句来分别输出两行结果。

【教师巡场指导】

  • 关注学生是否掌握基本结构、cout用法、分号;。
  • 提醒学生注意区分代码中的英文符号和中文符号。
  • 对于卡住的学生，引导他们查看"代码锦囊"或进行小组讨论。
  • 鼓励完成基础任务的学生尝试进阶挑战。

4. 成果展示与互评 (15分钟)

【小组/个人展示】(邀请2-3位同学/小组)

  • 请选择你完成得最好的一个任务（特别是任务4或任务5）。
  • 展示你的代码和运行结果。
  • 简单说明你是如何实现输出的，特别是注释是怎么写的，或者任务4/任务5的计算逻辑。

【互评与提问】

  • 其他同学可以提问，例如："你的注释写得很清楚，我一下就看懂了！"或者"你任务4是怎么做到一行输出那么多内容的？"
  • "点赞"环节：大家可以为你认为代码写得整洁、注释清晰、或者想法有创意的同学"点赞"。

【教师反馈】

  • 针对展示的代码，指出优点（例如：结构清晰、注释规范、输出了有趣的内容）。
  • 纠正共性错误（例如：忘记分号、main写错、中英文符号混用）。
  • 强调代码规范和注释的重要性。

5. 总结与延伸 (5分钟)

【知识点回顾】(可以用思维导图或列表方式)

  • C++是什么？（与计算机沟通的语言）
  • 第一个C++程序结构：
    • #include <bits/stdc++.h> (引入工具箱)
    • using namespace std; (使用标准工具)
    • int main() { ... return 0; } (主函数入口和结束)
  • 核心指令：cout << ... << endl; (输出内容并换行)
  • 基本计算：+ (加法), - (减法)
  • 重要笔记：// 注释内容 (给程序员看的解释)
  • 关键符号：{} (代码块边界), ; (语句结束标志)

【课后实践】(阶梯式作业)

  • 基础：确保能独立写出输出自己名字和年龄的程序，并添加注释。
  • 进阶：尝试使用C++进行更多的加减法运算，例如计算三个数的和 a + b + c。
  • 挑战：预习一下乘法 * 和除法 / 运算符，尝试用cout输出乘法或除法的结果。

【预告下节课】 下一节课，我们将更深入地学习cout的更多用法，并认识C++中一个非常重要的概念——变量！它就像一个可以存放数据的魔法盒子，让我们的程序更加灵活和强大！敬请期待！🎉
