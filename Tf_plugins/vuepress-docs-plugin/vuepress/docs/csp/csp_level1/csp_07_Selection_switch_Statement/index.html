🚉 从if到switch：多路选择的进阶

【故事开始】 小明的游戏已经使用if语句实现了基本的选择功能，玩家可以根据不同的情况看到不同的游戏内容。但很快他遇到了新问题：游戏里有一个"指令菜单"，玩家可以输入数字1-7选择不同的操作（比如1是攻击，2是防御，3是使用道具等）。小明用if-else if写了很长的代码，看起来很乱。小红看到后说："这种情况下可以用switch语句，它就像一个多向路口的交通指挥官，可以让程序代码更加清晰！"让我们和小明一起学习这个新的魔法工具吧！

在上一章中，我们学习了如何使用if语句让程序根据条件做出决策。if语句非常灵活，可以处理各种复杂的条件判断。但是，当我们需要根据一个变量的不同值执行不同的操作时，如果使用多个if-else if语句，代码可能会变得很长很乱。

想象一下，如果你需要根据用户输入的数字（1-7）显示对应的星期几，使用if-else if语句可能会这样写：

if (day == 1) {
    cout << "星期一" << endl;
} else if (day == 2) {
    cout << "星期二" << endl;
} else if (day == 3) {
    cout << "星期三" << endl;
// ... 以此类推
}

这时，switch语句就派上用场了！

![switch语句就像交通指挥官，根据变量的值将程序引导到不同的路径]

【生活类比】 想象switch语句就像一个火车站的道岔系统，根据列车（变量）的目的地（值），将它导向不同的轨道（代码块）。这种结构更加清晰，也更容易维护。

🎛️ switch语句的基本结构

switch语句的基本结构如下：

switch (表达式) {
    case 值1:
        // 如果表达式等于值1，执行这里的代码
        break;
    case 值2:
        // 如果表达式等于值2，执行这里的代码
        break;
    case 值3:
        // 如果表达式等于值3，执行这里的代码
        break;
    // 可以有更多的case...
    default:
        // 如果表达式不等于任何case值，执行这里的代码
        break;
}

【编程思维】 switch语句的执行流程：

  1. 计算switch后面括号内表达式的值
  2. 将这个值与每个case后面的值比较
  3. 如果找到匹配的case，执行该case下面的代码
  4. 遇到break语句时，跳出整个switch结构
  5. 如果没有匹配的case，执行default下面的代码（如果有default的话）

让我们看一个例子，根据用户输入的数字显示对应的星期几：

#include <bits/stdc++.h>
using namespace std;

int main() {
    int day;
    
    cout << "请输入一个数字 (1-7): ";
    cin >> day;
    
    // 使用switch语句
    switch (day) {
        case 1:
            cout << "星期一" << endl;
            break;
        case 2:
            cout << "星期二" << endl;
            break;
        case 3:
            cout << "星期三" << endl;
            break;
        case 4:
            cout << "星期四" << endl;
            break;
        case 5:
            cout << "星期五" << endl;
            break;
        case 6:
            cout << "星期六" << endl;
            break;
        case 7:
            cout << "星期日" << endl;
            break;
        default:
            cout << "无效的数字！" << endl;
    }
    
    return 0;
}

【记忆助手】 记住switch语句结构： "switch里放表达式，case值要一一写， 代码执行完要断路，break语句不能忘， 若都不匹配找默认，default处理特殊情况。"

🚨 break语句的重要性

在每个case后面，我们都使用了break语句。这个语句非常重要，它告诉程序"这个case已经处理完毕，可以跳出switch结构了"。

【常见错误】 如果忘记了break，程序会继续执行下一个case中的代码，这称为"贯穿"（fall-through）。有时这可能是我们想要的效果，但大多数情况下这是一个错误。

看看下面的例子，这个错误很容易发生：

switch (option) {
    case 1:
        cout << "你选择了选项1" << endl;
        // 忘记了break！
    case 2:
        cout << "你选择了选项2" << endl;
        break;
}

如果option是1，程序会输出：

你选择了选项1
你选择了选项2

这可能不是我们想要的结果！

【有意的贯穿】 不过，有时我们确实希望多个case共享相同的代码。例如：

#include <bits/stdc++.h>
using namespace std;

int main() {
    int level = 2;
    
    cout << "你的游戏等级是 " << level << "，你获得了以下奖励：" << endl;
    
    switch (level) {
        case 3:
            cout << "- 一把传说中的剑" << endl;
            // 注意：这里没有break，是故意的！
        case 2:
            cout << "- 一套魔法盔甲" << endl;
            // 注意：这里没有break，是故意的！
        case 1:
            cout << "- 10个金币" << endl;
            break;
        default:
            cout << "无效的游戏等级！" << endl;
            break;
    }
    
    return 0;
}

在这个例子中，如果level是2，程序会输出：

你的游戏等级是 2，你获得了以下奖励：
- 一套魔法盔甲
- 10个金币

这是因为程序执行了case 2中的代码，然后由于没有break，继续执行了case 1中的代码。这种"贯穿"效果在这个例子中是有用的，因为它表示更高等级的玩家可以获得所有低等级的奖励。

【真实场景】 游戏中的道具解锁系统经常使用这种结构。例如，一个角色升级系统可能会有不同的技能随等级解锁，并且高等级的角色同时拥有低等级的所有技能。

🔄 多个case共用一个代码块

如果多个case需要执行相同的代码，我们可以将它们放在一起：

#include <bits/stdc++.h>
using namespace std;

int main() {
    int day;
    
    cout << "请输入一个数字 (1-7): ";
    cin >> day;
    
    // 使用switch语句
    switch (day) {
        case 1:
        case 2:
        case 3:
        case 4:
        case 5:
            cout << "今天是工作日，加油工作/学习！" << endl;
            break;
        case 6:
        case 7:
            cout << "今天是周末，好好放松一下吧！" << endl;
            break;
        default:
            cout << "输入错误，请输入1-7之间的数字。" << endl;
    }
    
    return 0;
}

在这个例子中，如果day的值是1、2、3、4或5，程序会输出"今天是工作日..."；如果是6或7，程序会输出"今天是周末..."。

【编程最佳实践】 这种将多个case放在一起的技术可以使代码更加简洁，减少重复，特别是当多个值需要执行相同的操作时。这也是switch语句相比if-else语句的一个优势。

🚫 switch语句的限制

switch语句虽然很有用，但它也有一些限制：

  1. switch表达式必须是整数类型（包括char）或枚举类型
  2. case标签必须是常量表达式，不能是变量或函数调用
  3. 不能在case标签中使用浮点数、字符串或布尔值

例如，下面的代码是有效的：

char grade = 'A';
switch (grade) {
    case 'A': cout << "优秀！" << endl; break;
    case 'B': cout << "良好！" << endl; break;
    // ...
}

但下面的代码是无效的：

double score = 95.5;
switch (score) {  // 错误：switch表达式不能是浮点数
    // ...
}

string name = "Alice";
switch (name) {  // 错误：switch表达式不能是字符串
    // ...
}

int x = 5;
switch (x) {
    case x + 1:  // 错误：case标签必须是常量表达式
        // ...
        break;
}

📊 常见问题解决方案

1. 计算星期几

一个常见的问题是计算某一天的后几天是星期几。例如，如果今天是星期三，那么后天就是星期五。我们可以使用switch语句来解决这个问题：

#include <bits/stdc++.h>
using namespace std;

int main() {
    int today;
    
    cout << "今天是星期几？（1-7，其中1代表星期一）：";
    cin >> today;
    
    // 计算后天是星期几
    int dayAfterTomorrow = today + 2;
    
    // 如果超过7，需要循环回到星期一
    if (dayAfterTomorrow > 7) {
        dayAfterTomorrow -= 7;
    }
    
    cout << "后天是星期";
    
    switch (dayAfterTomorrow) {
        case 1:
            cout << "一" << endl;
            break;
        case 2:
            cout << "二" << endl;
            break;
        case 3:
            cout << "三" << endl;
            break;
        case 4:
            cout << "四" << endl;
            break;
        case 5:
            cout << "五" << endl;
            break;
        case 6:
            cout << "六" << endl;
            break;
        case 7:
            cout << "日" << endl;
            break;
    }
    
    return 0;
}

【数学连接】 这个例子用到了"模运算"的概念。当我们计算后天的星期时，如果结果超过7，我们需要减去7来得到正确的星期几。这就像时钟上的数字，当超过12时，我们会回到1开始计数。

2. 计算月份天数

另一个常见问题是计算一个月有多少天。我们可以使用switch语句结合闰年判断来解决这个问题：

#include <bits/stdc++.h>
using namespace std;

int main() {
    int year, month;
    
    cout << "请输入年份和月份（用空格分隔）：";
    cin >> year >> month;
    
    int days;
    
    switch (month) {
        case 1: case 3: case 5: case 7: case 8: case 10: case 12:
            days = 31;
            break;
        case 4: case 6: case 9: case 11:
            days = 30;
            break;
        case 2:
            // 闰年判断：能被4整除但不能被100整除，或者能被400整除
            if ((year % 4 == 0 && year % 100 != 0) || (year % 400 == 0)) {
                days = 29;  // 闰年2月有29天
            } else {
                days = 28;  // 平年2月有28天
            }
            break;
        default:
            cout << "无效的月份！请输入1-12之间的数字。" << endl;
            return 1;  // 返回非零值表示程序异常结束
    }
    
    cout << year << "年" << month << "月有" << days << "天。" << endl;
    
    return 0;
}

【跨学科知识】 这个例子不仅涉及编程，还涉及日历知识。不同月份的天数需要记忆，而闰年规则（能被4整除但不能被100整除，或能被400整除）是一个历法知识点。这就是编程的魅力之一：它可以帮助我们实践和应用各种学科的知识！

3. 简单计算器

我们可以使用switch语句来实现一个简单的计算器：

#include <bits/stdc++.h>
using namespace std;

int main() {
    double num1, num2;
    char operation;
    
    cout << "简易计算器" << endl;
    cout << "请输入一个算式（如：3 + 5）：";
    cin >> num1 >> operation >> num2;
    
    switch (operation) {
        case '+':
            cout << num1 << " + " << num2 << " = " << num1 + num2 << endl;
            break;
        case '-':
            cout << num1 << " - " << num2 << " = " << num1 - num2 << endl;
            break;
        case '*':
            cout << num1 << " * " << num2 << " = " << num1 * num2 << endl;
            break;
        case '/':
            if (num2 != 0) {
                cout << num1 << " / " << num2 << " = " << num1 / num2 << endl;
            } else {
                cout << "错误：除数不能为零！" << endl;
            }
            break;
        default:
            cout << "不支持的运算符！请使用 +、-、* 或 /。" << endl;
            break;
    }
    
    return 0;
}

【错误处理】 注意在除法运算中，我们特别检查了除数是否为零。这是一个典型的错误处理实例，因为在数学中，除以零是未定义的操作。良好的程序应该预测可能的错误并妥善处理它们。

4. 季节判断

我们可以使用switch语句来根据月份判断季节：

#include <bits/stdc++.h>
using namespace std;

int main() {
    int month;
    
    cout << "请输入月份（1-12）：";
    cin >> month;
    
    cout << "这个月是";
    
    switch (month) {
        case 3: case 4: case 5:
            cout << "春季" << endl;
            break;
        case 6: case 7: case 8:
            cout << "夏季" << endl;
            break;
        case 9: case 10: case 11:
            cout << "秋季" << endl;
            break;
        case 12: case 1: case 2:
            cout << "冬季" << endl;
            break;
        default:
            cout << "无效的月份！请输入1-12之间的数字。" << endl;
            break;
    }
    
    return 0;
}

【真实场景】 商店的季节性商品展示可能会使用类似的逻辑。根据当前月份，系统自动调整网站首页推荐的商品：春季展示花卉和春装，夏季展示游泳装备和防晒产品，秋季展示学习用品和秋装，冬季展示滑雪装备和冬衣。

🤔 何时使用switch，何时使用if-else

  • 当你需要根据一个变量的不同值执行不同的操作，并且这些值是离散的（如1、2、3或'A'、'B'、'C'），使用switch语句通常更清晰。
  • 当你需要检查范围（如score >= 90）或复杂条件（如age >= 18 && hasID），使用if-else语句更合适。

【决策表】 选择switch还是if-else？

情况推荐使用的结构原因
检查一个变量是否等于几个离散值之一switch代码更清晰，维护更容易
检查值的范围if-elseswitch不能直接处理范围
检查多个条件组合if-else逻辑运算符可以组合条件
检查浮点数、字符串或布尔值if-elseswitch不支持这些类型
简单的菜单选择系统switch每个选项对应一个case，直观明了

🎯 挑战任务

【迷你项目：简单菜单系统】 创建一个简单的学生信息管理菜单系统，让用户可以选择不同的操作：

#include <bits/stdc++.h>
using namespace std;

int main() {
    string name = "小明";
    int age = 12;
    double score = 95.5;
    char grade = 'A';
    
    int choice;
    
    do {
        // 显示菜单
        cout << "\n===== 学生信息管理 =====" << endl;
        cout << "1. 查看学生信息" << endl;
        cout << "2. 修改学生姓名" << endl;
        cout << "3. 修改学生年龄" << endl;
        cout << "4. 修改学生分数" << endl;
        cout << "5. 计算并更新学生等级" << endl;
        cout << "6. 退出系统" << endl;
        cout << "请选择操作 (1-6): ";
        cin >> choice;
        
        // 使用switch处理用户选择
        switch (choice) {
            case 1:
                cout << "\n学生信息：" << endl;
                cout << "姓名：" << name << endl;
                cout << "年龄：" << age << "岁" << endl;
                cout << "分数：" << score << endl;
                cout << "等级：" << grade << endl;
                break;
                
            case 2:
                cout << "请输入新的姓名：";
                cin >> name;
                cout << "姓名已更新为：" << name << endl;
                break;
                
            case 3:
                cout << "请输入新的年龄：";
                cin >> age;
                cout << "年龄已更新为：" << age << "岁" << endl;
                break;
                
            case 4:
                cout << "请输入新的分数：";
                cin >> score;
                cout << "分数已更新为：" << score << endl;
                break;
                
            case 5:
                // 根据分数计算等级
                if (score >= 90) {
                    grade = 'A';
                } else if (score >= 80) {
                    grade = 'B';
                } else if (score >= 70) {
                    grade = 'C';
                } else if (score >= 60) {
                    grade = 'D';
                } else {
                    grade = 'F';
                }
                cout << "等级已更新为：" << grade << endl;
                break;
                
            case 6:
                cout << "感谢使用学生信息管理系统，再见！" << endl;
                break;
                
            default:
                cout << "无效的选择！请输入1-6之间的数字。" << endl;
        }
        
    } while (choice != 6);  // 当用户选择6时退出循环
    
    return 0;
}

【真实应用】 这种菜单驱动的界面在许多软件系统中非常常见，尤其是在命令行界面中。例如，ATM机的操作菜单、图书馆借书系统、餐厅点餐系统等都使用类似的菜单结构。虽然现代软件通常使用图形界面，但背后的逻辑仍然是类似的菜单选择系统。
