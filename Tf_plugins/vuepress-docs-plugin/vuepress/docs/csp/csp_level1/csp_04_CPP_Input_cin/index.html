📥 基本输入

cin是C++中用于接收用户输入的工具，它的工作方式与cout相反。如果说cout是计算机的"嘴巴"，那么cin就是计算机的"耳朵"。

我们使用>>符号（称为提取运算符）从cin中获取输入：

#include <bits/stdc++.h>
using namespace std;

int main() {
    string name;
    
    cout << "请输入你的名字：";
    cin >> name;
    
    cout << "你好，" << name << "！欢迎来到C++的世界！" << endl;
    
    return 0;
}
运行过程

当你运行这个程序时：

  1. 程序会显示"请输入你的名字："
  2. 程序会等待你输入内容并按下回车键
  3. 你输入的内容会被存储在name变量中
  4. 程序会使用你输入的名字向你打招呼

编程思维

输入过程的步骤：

  1. 创建一个变量来存储用户的输入
  2. 提示用户输入内容（使用cout）
  3. 使用cin和>>从用户那里获取输入
  4. 将输入存储到变量中
  5. 使用这个变量进行后续操作

【记忆助手】 记住cin和cout的方向： "cin向内，箭头朝内 (>>)， cout向外，箭头朝外 (<<)， 方向指向数据流动的去向。"

常见错误

  1. 数据类型不匹配 问题：当你期望输入一个数字，但用户输入了文字

    int age;
cout << "请输入年龄：";
cin >> age;  // 如果用户输入"十二"而不是"12"，程序可能会出错

  2. 忘记提示用户 ❌ cin >> name; // 用户不知道该输入什么 ✅ cout << "请输入姓名："; cin >> name; // 清晰地提示用户

真实场景

学校注册系统就是使用类似的输入方式工作的！当新生入学时，系统会要求输入姓名、年龄、班级等信息，然后存储这些数据并生成学生档案。通过这种方式，同一个程序可以处理不同学生的信息，而不需要为每个学生编写单独的代码。

🔄 连续输入多个值

你可以在一行中输入多个值，用空格分隔：

#include <bits/stdc++.h>
using namespace std;

int main() {
    int num1, num2, num3;
    
    cout << "请输入三个整数，用空格分隔：";
    cin >> num1 >> num2 >> num3;
    
    int sum = num1 + num2 + num3;
    double average = sum / 3.0;
    
    cout << "这三个数的总和是：" << sum << endl;
    cout << "这三个数的平均值是：" << average << endl;
    
    return 0;
}
示例输入和输出

输入：10 20 30 输出：

这三个数的总和是：60
这三个数的平均值是：20

程序会将10存储在num1中，20存储在num2中，30存储在num3中。

【数学连接】 计算多个数的平均值是数学中的常见操作。在C++中，我们可以轻松地实现这一点：先计算总和，然后除以数字的个数。注意这里使用了3.0而不是3进行除法，是为了得到小数结果而不是整数结果。

📝 输入包含空格的文本

使用普通的cin >>无法正确读取包含空格的文本，因为空格会被视为分隔符。如果你想输入完整的句子，可以使用==getline()==函数：

使用cin（不能读取空格）
#include <bits/stdc++.h>
using namespace std;

int main() {
    string name;
    
    cout << "请输入你的全名：";
    cin >> name;  // 只能读取到第一个空格前的内容
    
    cout << "你好，" << name << "！" << endl;
    
    return 0;
}
使用getline（可以读取空格）
#include <bits/stdc++.h>
using namespace std;

int main() {
    string fullName;
    string favoriteMovie;
    
    cout << "请输入你的全名：";
    getline(cin, fullName);
    
    cout << "请输入你最喜欢的电影：";
    getline(cin, favoriteMovie);
    
    cout << "\n" << fullName << "，你最喜欢的电影是《" << favoriteMovie << "》，真是个不错的选择！" << endl;
    
    return 0;
}
比较说明

cin >> vs getline(cin, ...)：

  • cin >> 读取到空格或回车为止
  • getline() 读取整行文本，包括空格

生活类比

想象你在填写一份表格：

  • 使用cin >>就像在表格的每个框中只能写一个单词
  • 使用getline()就像有一整行可以自由书写

⚠️ 混合使用cin和getline的注意事项

如果你在同一个程序中混合使用cin >>和getline()，可能会遇到一些问题。这是因为cin >>在读取输入后，会将回车键留在输入缓冲区中，而getline()会读取这个回车并认为已经读取了一行。

解决方法

在cin >>和getline()之间添加cin.ignore()来清除输入缓冲区中的回车：

#include <bits/stdc++.h>
using namespace std;

int main() {
    int age;
    string description;
    
    cout << "请输入你的年龄：";
    cin >> age;
    
    // 清除输入缓冲区中的回车
    cin.ignore(10000, '\n');
    
    cout << "请描述一下你自己：";
    getline(cin, description);
    
    cout << "你今年" << age << "岁。" << endl;
    cout << "关于你自己：" << description << endl;
    
    return 0;
}

生活类比

这就像你在超市结账时：

  • cin >>扫描了你的商品但没处理收据
  • cin.ignore()把收据清理掉
  • 然后getline()才能开始处理下一个顾客的商品

常见错误

混合使用cin >>和getline()但忘记使用cin.ignore()：

int score;
string comment;
cout << "请输入分数：";
cin >> score;
cout << "请输入评语：";
getline(cin, comment);  // 可能会直接跳过这一步！

解决方案：在cin和getline之间添加cin.ignore(10000, '\n');

🔍 输入验证

在实际应用中，用户可能会输入错误的数据。例如，当你要求输入年龄时，用户可能会输入负数或非数字字符。为了使程序更加健壮，我们需要验证用户的输入：

简单验证
#include <bits/stdc++.h>
using namespace std;

int main() {
    int age;
    
    cout << "请输入你的年龄（0-120岁）：";
    cin >> age;
    
    // 验证年龄是否在合理范围内
    if (age < 0 || age > 120) {
        cout << "输入的年龄不合理！" << endl;
    } else {
        cout << "你的年龄是" << age << "岁。" << endl;
    }
    
    return 0;
}
循环验证
#include <bits/stdc++.h>
using namespace std;

int main() {
    int age;
    bool validInput = false;
    
    // 循环直到获得有效输入
    while (!validInput) {
        cout << "请输入你的年龄（0-120岁）：";
        cin >> age;
        
        if (cin.fail() || age < 0 || age > 120) {
            cout << "输入无效，请重新输入一个0到120之间的整数。" << endl;
            cin.clear();  // 清除错误状态
            cin.ignore(10000, '\n');  // 清除输入缓冲区
        } else {
            validInput = true;  // 输入有效，退出循环
        }
    }
    
    cout << "你的年龄是" << age << "岁。" << endl;
    
    return 0;
}

输入验证的重要性

在实际应用程序中，输入验证是非常重要的。它可以：

  1. 防止程序因无效输入而崩溃
  2. 确保数据的准确性和有效性
  3. 提高用户体验，及时给予用户反馈

🧮 使用输入创建简单计算器

现在，让我们结合所学的知识，创建一个简单的计算器程序：

#include <bits/stdc++.h>
using namespace std;

int main() {
    double num1, num2;
    char operation;
    
    cout << "===== 简易计算器 =====" << endl;
    cout << "请输入第一个数：";
    cin >> num1;
    
    cout << "请输入运算符（+、-、*、/）：";
    cin >> operation;
    
    cout << "请输入第二个数：";
    cin >> num2;
    
    // 根据运算符执行相应的计算
    switch (operation) {
        case '+':
            cout << num1 << " + " << num2 << " = " << num1 + num2 << endl;
            break;
        case '-':
            cout << num1 << " - " << num2 << " = " << num1 - num2 << endl;
            break;
        case '*':
            cout << num1 << " * " << num2 << " = " << num1 * num2 << endl;
            break;
        case '/':
            if (num2 != 0) {
                cout << num1 << " / " << num2 << " = " << num1 / num2 << endl;
            } else {
                cout << "错误：除数不能为零！" << endl;
            }
            break;
        default:
            cout << "错误：无效的运算符！" << endl;
    }
    
    return 0;
}
示例运行

输入：

请输入第一个数：10
请输入运算符（+、-、*、/）：*
请输入第二个数：5

输出：

10 * 5 = 50

程序分析

这个计算器程序展示了如何：

  1. 使用cin获取用户输入的数字和运算符
  2. 使用条件语句（switch-case）根据不同的运算符执行不同的操作
  3. 处理特殊情况（如除以零）
  4. 将计算结果输出给用户
