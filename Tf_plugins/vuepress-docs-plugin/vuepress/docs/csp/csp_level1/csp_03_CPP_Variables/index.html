🧸 什么是变量？

学习目标

在本章结束后，你将能够：

  • 理解变量的概念和重要性
  • 创建和使用不同类型的变量
  • 掌握变量命名的规则和最佳实践
  • 对变量进行各种运算操作
【故事开始】

小明想制作一个简单的游戏，记录玩家的分数、血量和等级。但他发现每次数据变化时，都要修改代码中的具体数字，太麻烦了！老师告诉他可以使用"变量"来解决这个问题。变量就像是带标签的魔法盒子，可以存放会变化的数据。让我们和小明一起学习变量的魔法吧！

想象一下，变量就像是一个带标签的盒子，你可以在里面存放各种东西，比如数字、文字或其他信息。每当你需要使用这些信息时，只需要通过盒子的标签（变量名）就能找到它们！

重要概念

在编程中，变量让我们能够存储和管理数据，这样我们就可以在程序的不同部分使用这些数据。

快速检查

变量的主要作用是什么？

  • A. 计算数学公式
  • B. 输出信息到屏幕
  • C. 存储和管理数据
  • D. 连接到互联网
查看答案

答案是 C. 存储和管理数据

📝 如何创建变量

在C++中创建变量非常简单，你需要指定两件事：

  1. 变量的类型（你要存放什么类型的数据）
  2. 变量的名字（你给这个盒子贴什么标签）

变量声明语法

基本语法是：

类型 变量名;          // 声明变量
类型 变量名 = 初始值;  // 声明并初始化变量
基本变量示例
#include <bits/stdc++.h>
using namespace std;

int main() {
    // 创建一个整数变量
    int age = 12;
    
    // 创建一个小数变量
    double height = 1.58;
    
    // 创建一个字符变量
    char grade = 'A';
    
    // 创建一个字符串变量
    string name = "小明";
    
    // 输出这些变量
    cout << "姓名：" << name << endl;
    cout << "年龄：" << age << "岁" << endl;
    cout << "身高：" << height << "米" << endl;
    cout << "成绩：" << grade << endl;
    
    return 0;
}
输出结果
姓名：小明
年龄：12岁
身高：1.58米
成绩：A

编程思维

定义变量的步骤：

  1. 决定你需要存储什么类型的数据
  2. 为这个数据选择一个有意义的名字
  3. 使用正确的语法声明变量
  4. (可选) 给变量赋初始值

常见错误

  1. 使用变量前忘记声明 ❌ score = 95; // 如果score未声明，这会导致错误 ✅ int score = 95; // 先声明后使用

  2. 数据类型不匹配 ❌ int average = 95.5; // 95.5是小数，int存储整数 ✅ double average = 95.5; // 使用double存储小数

🔄 修改变量的值

变量的特性

变量最酷的地方在于，你可以随时改变它们的值！就像换掉盒子里的东西一样。

修改变量示例
#include <bits/stdc++.h>
using namespace std;

int main() {
    int score = 85;  // 初始分数
    
    cout << "原始分数：" << score << endl;
    
    // 加分！
    score = 95;
    
    cout << "更新后的分数：" << score << endl;
    
    // 再加分！
    score = score + 5;  // 也可以写成 score += 5;
    
    cout << "最终分数：" << score << endl;
    
    return 0;
}
输出结果
原始分数：85
更新后的分数：95
最终分数：100

真实场景

游戏中的计分系统就是使用变量实现的！想象一个"打地鼠"游戏：

  • 每打中一只地鼠，玩家的score变量就增加10分
  • 如果打中了金地鼠，score变量增加50分
  • 游戏可以随时显示最新的score值，而不需要重新计算总分

快速检查

执行以下代码后x的值是多少？

int x = 10;
x = x * 2;
x = x + 5;
  • A. 10
  • B. 20
  • C. 25
  • D. 30
查看答案

答案是 C. 25，因为：

  1. 初始值 x = 10
  2. x = x * 2 → x = 10 * 2 = 20
  3. x = x + 5 → x = 20 + 5 = 25

C++中有很多不同类型的变量，就像我们有不同类型的盒子来存放不同的物品：

常用变量类型表

类型用途例子

布尔类型

【数学连接】 变量类型与数学中的数字集合类似：

  • int类似于整数集合 Z = {-2, -1, 0, 1, 2, ...}
  • double类似于实数集合 R，可以表示带小数点的数
  • bool类似于集合 {0, 1} 或 {假, 真}，只有两个可能的值

🏷️ 变量命名规则

给变量起名字有一些规则，就像给宠物起名字一样，不能太奇怪：

变量命名规则
  1. 变量名只能包含字母、数字和下划线
  2. 变量名必须以字母或下划线开头
  3. 变量名区分大小写（age和Age是两个不同的变量）
  4. 变量名不能是C++的关键字（如int、double等）
好的变量名
// 好的变量名例子
int studentAge = 12;
double averageScore = 92.5;
string firstName = "小明";
bool hasCompletedHomework = true;
int _privateCounter = 0;
不好的变量名
// 不好的变量名例子
int a = 12;                  // 不够描述性
double avg = 92.5;           // 缩写可能不清晰
string n = "小明";            // 太短，不明确
bool h = true;               // 不知道这表示什么
int 1stPlace = 1;            // 不能以数字开头（非法）
int class = 5;               // 不能使用关键字（非法）

编程最佳实践

变量命名的建议：

  1. 使用有意义的名字，能反映变量的用途
  2. 对于由多个单词组成的变量名，使用驼峰命名法（如studentAge）或下划线分隔（如student_age）
  3. 保持一致的命名风格

快速检查

以下哪个变量名是非法的？

  • A. student_age
  • B. 1stPlace
  • C. userName
  • D. _count
查看答案

答案是 B. 1stPlace，因为变量名不能以数字开头

🔒 常量：不变的变量

重要概念

有时候，我们需要一些值在程序运行过程中永远不变，这时我们可以使用常量。常量就像是被锁起来的变量，一旦设定就不能改变。

常量示例
#include <bits/stdc++.h>
using namespace std;

int main() {
    // 定义常量
    const double PI = 3.14159;
    const int DAYS_IN_WEEK = 7;
    
    // 尝试修改常量会导致错误
    // PI = 3.14;  // 这行会导致编译错误！
    
    cout << "圆周率：" << PI << endl;
    cout << "一周有 " << DAYS_IN_WEEK << " 天" << endl;
    
    // 使用常量计算
    int radius = 5;
    double circumference = 2 * PI * radius;
    
    cout << "半径为 " << radius << " 的圆的周长是 " << circumference << endl;
    
    return 0;
}
输出结果
圆周率：3.14159
一周有 7 天
半径为 5 的圆的周长是 31.4159

真实场景

常量在科学计算中很常见。比如：

  • 当计算物体下落的速度时，我们需要使用重力加速度常量g = 9.8 m/s²
  • 当计算电子设备功率时，我们可能需要使用电压常量
  • 这些物理量在计算过程中通常不会改变

常量命名约定

常量通常使用全大写字母命名，多个单词之间用下划线分隔，如MAX_SCORE、PI、GAME_SPEED等。这种约定使得常量在代码中很容易被识别。

🧮 变量的运算

你可以对变量进行各种运算，就像使用计算器一样：

基本运算
#include <bits/stdc++.h>
using namespace std;

int main() {
    int a = 10;
    int b = 3;
    
    // 基本运算
    cout << "a + b = " << a + b << endl;  // 加法
    cout << "a - b = " << a - b << endl;  // 减法
    
    return 0;
}
乘除运算
#include <bits/stdc++.h>
using namespace std;

int main() {
    int a = 10;
    int b = 3;
    
    // 乘除运算
    cout << "a * b = " << a * b << endl;  // 乘法
    cout << "a / b = " << a / b << endl;  // 整数除法（结果为3，不是3.333...）
    
    // 使用double进行除法
    double c = 10.0;
    double d = 3.0;
    cout << "c / d = " << c / d << endl;  // 浮点数除法（结果为3.333...）
    
    return 0;
}
取余和复合运算
#include <bits/stdc++.h>
using namespace std;

int main() {
    int a = 10;
    int b = 3;
    
    // 取余运算
    cout << "a % b = " << a % b << endl;  // 取余（10除以3的余数是1）
    
    // 复合赋值运算符
    int x = 5;
    cout << "初始x值: " << x << endl;
    
    x += 3;  // 等同于 x = x + 3;
    cout << "x += 3 后: " << x << endl;
    
    x -= 2;  // 等同于 x = x - 2;
    cout << "x -= 2 后: " << x << endl;
    
    x *= 4;  // 等同于 x = x * 4;
    cout << "x *= 4 后: " << x << endl;
    
    x /= 2;  // 等同于 x = x / 2;
    cout << "x /= 2 后: " << x << endl;
    
    x %= 3;  // 等同于 x = x % 3;
    cout << "x %= 3 后: " << x << endl;
    
    return 0;
}
输出结果示例
a + b = 13
a - b = 7
a * b = 30
a / b = 3
c / d = 3.33333
a % b = 1
初始x值: 5
x += 3 后: 8
x -= 2 后: 6
x *= 4 后: 24
x /= 2 后: 12
x %= 3 后: 0

注意整数除法

在C++中，当两个整数相除时，结果也是整数，小数部分会被舍去。例如，10 / 3的结果是3，而不是3.333...。如果你想得到小数结果，至少有一个操作数必须是小数类型。

🔢 递增和递减运算符

C++提供了特殊的运算符来让变量加1或减1，这在计数和循环中非常有用：

递增递减示例
#include <bits/stdc++.h>
using namespace std;

int main() {
    int count = 5;
    
    // 递增运算符 ++
    count++;  // 等同于 count = count + 1;
    cout << "count++ 后: " << count << endl;
    
    // 递减运算符 --
    count--;  // 等同于 count = count - 1;
    cout << "count-- 后: " << count << endl;
    
    // 前置和后置的区别
    int a = 5;
    int b = 5;
    
    int resultA = ++a;  // 先增加a，再赋值给resultA
    int resultB = b++;  // 先赋值给resultB，再增加b
    
    cout << "++a 的结果: a = " << a << ", resultA = " << resultA << endl;
    cout << "b++ 的结果: b = " << b << ", resultB = " << resultB << endl;
    
    return 0;
}
输出结果
count++ 后: 6
count-- 后: 5
++a 的结果: a = 6, resultA = 6
b++ 的结果: b = 6, resultB = 5

前置与后置的区别

  • 前置递增/递减 (++i 或 --i): 先增加/减少变量的值，然后返回新值
  • 后置递增/递减 (i++ 或 i--): 先返回变量的当前值，然后增加/减少变量的值

挑战练习 🌟🌟🌟

创建一个简单的游戏状态系统，包含玩家的生命值、分数和等级，并模拟一些游戏事件（如获得分数、受到伤害、升级）

#include <bits/stdc++.h>
using namespace std;

int main() {
    // 初始化游戏状态
    int health = 100;
    int score = 0;
    int level = 1;
    const int POINTS_PER_LEVEL = 100;  // 每100分升一级
    
    cout << "===== 游戏开始 =====" << endl;
    cout << "生命值: " << health << ", 分数: " << score << ", 等级: " << level << endl;
    
    // 模拟游戏事件
    cout << "\n[事件] 玩家击败了一个敌人！" << endl;
    score += 20;
    cout << "生命值: " << health << ", 分数: " << score << ", 等级: " << level << endl;
    
    cout << "\n[事件] 玩家受到了伤害！" << endl;
    health -= 30;
    cout << "生命值: " << health << ", 分数: " << score << ", 等级: " << level << endl;
    
    cout << "\n[事件] 玩家获得了宝箱！" << endl;
    score += 50;
    cout << "生命值: " << health << ", 分数: " << score << ", 等级: " << level << endl;
    
    // 检查是否升级
    if (score >= POINTS_PER_LEVEL) {
        level++;
        cout << "\n恭喜！玩家升级了！" << endl;
        // 升级奖励：恢复生命值
        health += 20;
        if (health > 100) health = 100;  // 生命值上限为100
    }
    
    cout << "\n===== 当前状态 =====" << endl;
    cout << "生命值: " << health << ", 分数: " << score << ", 等级: " << level << endl;
    
    return 0;
}

迷你项目：温度转换器

创建一个温度转换程序，可以将摄氏度转换为华氏度，或者将华氏度转换为摄氏度。

转换公式：

  • 华氏度 = 摄氏度 × 9/5 + 32
  • 摄氏度 = (华氏度 - 32) × 5/9
#include <bits/stdc++.h>
using namespace std;

int main() {
    // 定义变量
    double celsius = 25.0;       // 摄氏度
    double fahrenheit;           // 华氏度
    
    // 摄氏度转华氏度
    fahrenheit = celsius * 9.0/5.0 + 32;
    
    cout << celsius << " 摄氏度 = " << fahrenheit << " 华氏度" << endl;
    
    // 华氏度转摄氏度
    celsius = (fahrenheit - 32) * 5.0/9.0;
    
    cout << fahrenheit << " 华氏度 = " << celsius << " 摄氏度" << endl;
    
    return 0;
}

提示：确保在除法运算中使用小数（如9.0/5.0）以获得准确的结果。
