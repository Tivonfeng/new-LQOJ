枚举法

〽️ 什么是枚举法

枚举法也称作穷举法，指一一列举所有可能的答案。

啥意思呢？比如请问1-10中，所有的偶数有哪些？那么我们从1数到10，把是偶数的数字2、4、6、8一一写下来，这样解决问题的方法就叫做枚举法。

〽️ 枚举法的解题思路（步骤）

  1. 确定求解问题答案的范围

  2. 枚举（一一列举）所有可能的答案

  3. 从列举出所有可能的答案中，筛选出最终的答案

  4. （可选）考虑枚举和筛选过程的优化加速

〽️ 枚举法的优缺点

优点：

  • 思路不复杂，甚至可以说是简单，整个过程就是一一列举，然后选出符合条件的答案

缺点：

  • 数据量提升时，易产生超时现象

  • 为了解决超时，常规思路是借助数据结构、设置一些约束条件、数学公式运算等进行优化

4.1 经典找质数

信息学奥赛一本通：2040

http://ybt.ssoier.cn:8088/problem_show.php?pid=2040

【题目描述】

求出n(2≤n≤1000)以内的全部质数。

【输入格式】

输入n。

【输出格式】

输出包含多行：

由小到大的质数。

【样例输入】

10

【样例输出】

2

3

5

7

#include<bits/stdc++.h>
using namespace std;
//判断是不是质数 
//4. 可考虑优化（选做）
bool check(int num){
    for(int i = 2; i < num; i++){
        if(num % i == 0){
            return false;
        }
    }
    return true;
}

int main(){
    //1. 确定求解问题答案的范围 
    int n = 0;
    cin >> n;
    //2. 枚举所有可能的答案 
    for(int i = 2; i < n; i++){
        //3. 从列举出所有可能的答案中，筛选出最终的答案
        if(check(i)){
            cout << i << endl;
        }
    }
    return 0;
}

4.2 猜猜父亲今年多少岁

【题目描述】

小明的父亲说：”我的年龄是一个两位数，我的儿子比我小27岁。如果把我年龄的数字交换位置，恰好就是我儿子的年龄“。请问小明的父亲和小明所有可能的年龄是多少？

【输入格式】

无

【输出格式】

所有可能的父亲年龄和儿子年龄

输出包含多行：

后续多行，一行两个数字，第一个为父亲的年龄，第二个为儿子的年龄，用空格隔开

最后一行一个数字，表示总共多少种可能

【样例输入】

无

【样例输出】

无

#include<bits/stdc++.h>
using namespace std;
int main(){
    int one = 0;
    int ten = 0;
    int sonAge = 0;
    int count = 0;
    for(int fatherAge = 27; fatherAge < 100; fatherAge++){
        one = fatherAge % 10;
        ten = fatherAge / 10;
        if((fatherAge - 27) == (one * 10 + ten)){
            count++;
            sonAge = one * 10 + ten;
            cout << fatherAge << " " << sonAge << endl;
        }
    }
    cout << count;
    return 0;
}

4.3 数字拼接

【题目描述】

输入一个整数n（3≤n≤9），从0到n中任意选择三个整数（包括0和n），组成一个三位数（0不能作为百位），三位数各位数字不能重复，请问这个三位数是奇数有多少种可能？

【输入格式】

输入n。

【输出格式】

输出包含一行：

一行一个数字，表示总共多少种可能

【样例输入】

3

【样例输出】

8

#include<bits/stdc++.h>
using namespace std;
int main(){
    int n = 0;
    int count = 0;
    cin >> n;
    for(int i = 1; i <= n; i++){
        for(int j = 0; j <= n; j++){
            if(j != i){
                for(int k = 0; k <= n; k++){
                    if((k != i) && (k != j)){
                        if(k % 2 == 1){
                            count++;
                        }
                    }
                }
                //优化 
//              for(int k = 1; k <= n; k += 2){
//                  if((k != i) && (k != j)){
//                      count++;
//                  }
//              }
            }
        }
    }
    cout << count;
    return 0;
}

4.4 特殊的自然数

【题目描述】

一个十进制自然数，它的七进制和九进制表示都是三位数，且七进制表示倒过来就是九进制表示，请问十进制自然数是多少？

【输入格式】

无

【输出格式】

输出可能包含多行，依次输出十进制、七进制、九进制

【样例输入】

无

【样例输出】

无

#include<bits/stdc++.h>
using namespace std;
int main(){
    int sevenOne, sevenTen, sevenHundred;
    int nineOne, nineTen, nineHundred;
    for(int i = 81; i <= 342; i++){
        sevenOne = i % 7;
        sevenTen = (i / 7) % 7;
        sevenHundred = i / 7 / 7;
        nineOne = i % 9;
        nineTen = (i / 9) % 9;
        nineHundred = i / 9 / 9;
        if((sevenOne == nineHundred) && (sevenTen == nineTen) && (sevenHundred == nineOne)){
            cout << i << endl;

        }
    }

}

5.2 完数

【题目描述】

完数指除了其自身，所有其他的因子的和，输入一个整数n，找出n以内的所有的完数。

【输入格式】

输入n。

【输出格式】

输出包含多行：

一行一个数字，表示不同的完数。

【样例输入】

100

【样例输出】

6

28

#include<bits/stdc++.h>
using namespace std;
bool check(int num){
    int sum = 0; 
    for(int i = 1; i < num; i++){
        if(num % i == 0){
            sum += i;
        }
    }
    if(sum == num){
        return true;
    }
    return false;
}

int main(){
    int n = 0;
    cin >> n;
    for(int i = 2; i <= n; i++){
        if(check(i)){
            cout << i << endl;
        }
    }
    return 0;
}

//28 = 1 + 2 + 4 + 7 + 14

5.3 经典之水仙花数

【题目描述】

水仙花数是一个三位数，它的各位数字的三次方之和等于其自身，找出所有的水仙花数。

【输入格式】

无。

【输出格式】

输出包含多行：

一行一个数字，表示不同的水仙花数。

【样例输入】

无。

【样例输出】

153

370

371

407

#include<bits/stdc++.h>
using namespace std;
bool check(int num){
    int one = num % 10;
    int ten = ((num - one) / 10) % 10;
    int hundred = num / 100;
    if((pow(one, 3) + pow(ten, 3) + pow(hundred, 3)) == num){
        return true;
    }
    return false;
}

int main(){
    for(int i = 100; i < 1000; i++){
        if(check(i)){
            cout << i << endl;
        }
    }
    return 0;
}
