Streamlit UI组件教程

Streamlit 是一个强大的 Web GUI（网页交互）功能库，它提供了丰富的 UI 组件，可以帮助我们快速实现数据可视化和交互式应用开发。

安装和基础设置

pip install streamlit
streamlit run your_app.py

输出类组件

文本类输出组件

  1. st.write() - 通用输出
st.write("这是一段文本")
st.write(df)  # 显示数据框
  1. st.markdown() - Markdown 格式输出
st.markdown("# 标题")
st.markdown("**加粗文本**")
  1. st.code() - 代码块
st.code("print('Hello World')", language="python")
  1. st.latex() - LaTeX 公式
st.latex(r'''E=mc^2''')

数据展示组件

  1. st.dataframe() - 数据框显示
st.dataframe(df)
  1. st.table() - 静态表格
st.table(df)
  1. st.json() - JSON 数据
st.json({"key": "value"})

图表组件

  1. st.line_chart() - 折线图
st.line_chart(data)
  1. st.bar_chart() - 柱状图
st.bar_chart(data)
  1. st.area_chart() - 面积图
st.area_chart(data)
  1. st.map() - 地图
st.map(data)

进度和状态组件

  1. st.progress() - 进度条
progress = st.progress(0)
  1. st.spinner() - 加载动画
with st.spinner("加载中..."):
    time.sleep(2)
  1. st.success() - 成功提示
st.success("操作成功！")
  1. st.error() - 错误提示
st.error("发生错误！")
  1. st.warning() - 警告提示
st.warning("请注意！")
  1. st.info() - 信息提示
st.info("这是一条信息")

输入类组件

文本类输入组件

  1. st.text_input() - 单行文本输入
name = st.text_input("请输入您的姓名", "默认值")
  1. st.text_area() - 多行文本输入
feedback = st.text_area("请输入您的反馈", "默认反馈内容")

按键类输入组件

  1. st.button() - 按钮
if st.button("点击我"):
    st.write("按钮被点击了！")
  1. st.download_button() - 下载按钮
st.download_button("下载文件", data, file_name="example.txt")

选项类输入组件

  1. st.selectbox() - 下拉选择框
option = st.selectbox("选择一个选项", ["选项1", "选项2", "选项3"])
  1. st.multiselect() - 多选下拉框
options = st.multiselect("选择多个选项", ["选项1", "选项2", "选项3"])
  1. st.radio() - 单选按钮
choice = st.radio("选择一个选项", ["选项1", "选项2", "选项3"])
  1. st.checkbox() - 复选框
agree = st.checkbox("我同意")

时间类输入组件

  1. st.date_input() - 日期选择器
date = st.date_input("选择日期")
  1. st.time_input() - 时间选择器
time = st.time_input("选择时间")

数字类输入组件

  1. st.number_input() - 数字输入
number = st.number_input("输入一个数字", min_value=0, max_value=100, value=50)
  1. st.slider() - 滑块
value = st.slider("选择一个值", 0, 100, 50)

媒体类输入组件

  1. st.file_uploader() - 文件上传
uploaded_file = st.file_uploader("上传文件", type=["csv", "txt"])
  1. st.camera_input() - 摄像头输入
picture = st.camera_input("拍照")

布局组件

  1. st.columns() - 列布局
col1, col2 = st.columns(2)
with col1:
    st.write("第一列")
with col2:
    st.write("第二列")
  1. st.expander() - 可展开区域
with st.expander("点击展开"):
    st.write("展开的内容")
  1. st.container() - 容器
with st.container():
    st.write("容器中的内容")

缓存和性能优化

  1. @st.cache_data - 数据缓存
@st.cache_data
def load_data():
    return pd.read_csv("data.csv")
  1. @st.cache_resource - 资源缓存
@st.cache_resource
def load_model():
    return load_ml_model()

会话状态

if "counter" not in st.session_state:
    st.session_state.counter = 0

st.session_state.counter += 1
st.write(f"计数: {st.session_state.counter}")

st.set_page_config(
    page_title="我的应用",
    page_icon=":rocket:",
    layout="wide",
    initial_sidebar_state="expanded"
