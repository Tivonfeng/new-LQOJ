Python Level4

  1. AI智能聊天机器人

    • 技术栈：Python Streamlit库, Python, OpenAI API
    • 功能描述：实现智能对话系统，支持多轮对话、上下文理解，可设置不同性格和对话风格。

  2. AI多人聊天室

    • 技术栈：Python Streamlit库, Python, WebSocket, OpenAI API
    • 功能描述：支持多人实时聊天，包含AI助手参与对话，支持表情、图片发送，聊天记录保存。

  3. AI趣味对联生成器

    • 技术栈：Python Streamlit库, Python, OpenAI API
    • 功能描述：输入上联或关键词，AI生成下联和横批，支持选择对联风格（传统/现代/搞笑）。

  4. AI动物识别

    • 技术栈：Python Streamlit库, Python, 图像识别API
    • 功能描述：上传动物照片，识别动物种类，提供相关知识和趣味信息。

  5. AI表情包生成

    • 技术栈：Python Streamlit库, Python, 图像生成API
    • 功能描述：输入文字或选择模板，生成个性化表情包，支持自定义编辑。

  6. AI证件照生成

    • 技术栈：Python Streamlit库, Python, OpenCV, 图像处理API
    • 功能描述：上传照片，自动生成标准证件照，支持背景替换、尺寸调整。

  7. 扑克牌预测分析

    • 技术栈：Python Streamlit库, Python, 数据分析库
    • 功能描述：分析扑克牌游戏数据，预测胜率，提供策略建议。

  8. 小猪佩奇语言合成

    • 技术栈：Python Streamlit库, Python, 语音合成API
    • 功能描述：将文字转换为小猪佩奇风格的语音，支持语速、音调调整。

  9. AI智能体制作

    • 技术栈：Python Streamlit库, Python, OpenAI API, LangChain
    • 功能描述：提供可视化界面，让用户自定义AI智能体的性格、知识库和对话风格，支持智能体训练和部署。
