🌟 知识迁移：在Scratch中，我们只需要拖拽"说"模块，输入想说的话，角色就能开口说话了。那在Python中，我们怎么实现这个功能呢？今天老师要教大家一个超级厉害的指令 —— print！

趣味小知识

你知道吗？Python的print指令就像是一个会说话的小助手，它可以把我们想说的话告诉计算机，然后计算机就会在屏幕上显示出来！而且它比Scratch的"说"模块还要强大，不仅能显示文字，还能显示数字、计算结果，甚至能显示表情符号！是不是很神奇？🤩
