🛠️ 二、知识讲解（20分钟）

🧰 2.1 print基础语法

✨ 基本用法 ✨

# 输出不同类型的数据
print(1)          # 输出整数
print(1.2)        # 输出浮点数
print("hello")    # 输出字符串
print("hello", "world", "China")  # 输出多个数据

🧰 2.2 控制输出格式

💫 间隔符号控制 💫

# 默认空格分隔
print('hello', 'world', 'lqcode')  # 输出：hello world lqcode

# 自定义分隔符
print('hello', 'world', 'lqcode', sep='-')  # 输出：hello-world-lqcode

💫 末尾符号控制 💫

print('hello', end='-')
print('world')  # 输出：hello-world

🧰 2.3 格式化输出

逗号分隔
# 最简单的方式
name = 'tivonfeng'
print("这位同学的姓名是：", name)
加号拼接
# 字符串拼接
name = 'tivonfeng'
print("my name is " + name)  # 注意：加号不会自动添加空格
占位符
# 使用占位符
name = '冯老师'
sex = '男'
number = 123
height = 1.78
weight = 65
print('姓名是%s，性别是%s，学号是%d，身高是%fm，体重是%dkg' % (name, sex, number, height, weight))
f字符串
# 最现代的方式
name = '冯老师'
sex = '男'
number = 123
height = 1.78
weight = 65
print(f'姓名是{name}，性别是{sex}，学号是{number}，身高是{height}m，体重是{weight}kg')

🧰 2.4 数据宽度限制

占位符方式
name = '冯老师'
number = 123
height = 1.78
print('姓名是%8s，学号是%08d，身高是%0.4fm' % (name, number, height))
f字符串方式
name = '冯老师'
number = 123
height = 1.78
print(f'姓名是{name:8s}，学号是{number:08d}，身高是{height:0.2f}m')
