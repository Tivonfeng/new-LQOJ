# 学生数据分析插件

学生数据分析与统计插件，用于收集和分析学生在OJ平台上的学习行为数据。

## 功能特性

- 学习行为数据收集
- 学习数据分析与统计
- 可视化数据展示
- 管理员数据分析面板

## 数据来源分析

详细的数据分析总结请参考：[学生数据分析总结.md](./学生数据分析总结.md)

该文档基于 [Hydro 数据库结构文档](https://hydro.js.org/docs/Hydro/dev/db-layout) 分析，总结了可以从 Hydro 系统中提炼的以下数据：

### 主要数据源
1. **提交记录数据** (record 集合) - 提交行为、判题结果、代码质量
2. **题目状态数据** (document.status) - 题目完成情况、收藏行为
3. **题目统计数据** (record.stat) - 最佳记录、性能指标
4. **题目信息数据** (document) - 题目元数据、难度、标签
5. **比赛数据** (document - Tdoc) - 比赛参与、排名、表现
6. **训练数据** (document - TrainingDoc) - 训练进度、知识点掌握
7. **用户行为数据** (oplog) - 访问路径、操作记录、活跃度
8. **访问频率数据** (opcount) - 操作频率、功能使用
9. **用户基础信息** (user) - 用户元数据、注册登录信息

### 可分析维度
- 学习行为分析（提交频率、解决模式、代码质量）
- 学习效果分析（能力评估、进步轨迹、瓶颈识别）
- 学习偏好分析（内容偏好、时间偏好）
- 社交与竞争分析（比赛表现、社区参与）

## 安装

将插件目录放置到 `Tf_plugins/student-analytics` 目录下。

## 配置

在系统配置中启用插件：

```yaml
student-analytics:
  enabled: true
```

## 使用

访问 `/analytics/student` 查看学生数据分析页面。

管理员可访问 `/analytics/student/admin` 查看全局数据分析。
